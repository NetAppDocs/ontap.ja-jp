= ONTAP Vscanスキャナ プールの設定について学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
スキャナ プールは、SVMに接続できるVscanサーバと特権ユーザを定義します。スキャナ ポリシーは、スキャナ プールがアクティブかどうかを定義します。

[NOTE]
====
SMBサーバでエクスポート ポリシーを使用する場合は、各Vscanサーバをエクスポート ポリシーに追加する必要があります。

====
