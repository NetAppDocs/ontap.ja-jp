= スキャナプールの概要の設定
:icons: font
:imagesdir: ../media/


[role="lead"]
スキャナプールは、 SVM に接続できる Vscan サーバと特権ユーザを定義します。スキャナポリシーは、スキャナプールがアクティブかどうかを決定します。

[NOTE]
====
CIFS サーバでエクスポートポリシーを使用する場合は、各 Vscan サーバをエクスポートポリシーに追加する必要があります。

====
