= ONTAP Vscanスキャナプールの設定について学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
スキャナプールは、SVMに接続できるVscanサーバと特権ユーザを定義します。スキャナポリシーは、スキャナプールがアクティブかどうかを決定します。

[NOTE]
====
SMBサーバでエクスポートポリシーを使用する場合は、各Vscanサーバをエクスポートポリシーに追加する必要があります。

====
