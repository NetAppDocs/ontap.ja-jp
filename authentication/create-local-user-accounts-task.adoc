= ローカルアカウントアクセスの有効化の概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ローカルアカウントでは、アカウント情報、公開鍵、セキュリティ証明書がストレージシステムに格納されます。「 securitylogin create 」コマンドを使用すると、ローカルアカウントが管理 SVM またはデータ SVM にアクセスできるようになります。
