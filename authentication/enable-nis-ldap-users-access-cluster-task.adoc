= LDAP サーバまたは NIS サーバのアクセスの概要を設定
:icons: font
:imagesdir: ../media/


[role="lead"]
LDAP アカウントまたは NIS アカウントから SVM にアクセスするためには、 LDAP サーバまたは NIS サーバから SVM へのアクセスを設定しておく必要があります。スイッチ機能を使用すると、 LDAP または NIS を代替ネームサービスソースとして使用できます。
