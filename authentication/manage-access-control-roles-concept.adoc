= アクセス制御ロールの概要
:icons: font
:imagesdir: ../media/


[role="lead"]
管理者がアクセスできるコマンドは、管理者に割り当てられたロールで決まります。ロールは管理者のアカウントを作成するときに割り当てます。必要に応じて、別のロールを割り当てたりカスタムロールを定義したりできます。
