= ONTAP管理者アカウントの管理について
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アカウントアクセスを有効にした方法によっては、ローカルアカウントへの公開鍵の関連付け、 CA 署名済みサーバデジタル証明書のインストール、 AD 、 LDAP 、 NIS のアクセスの設定などが必要になる場合があります。これらのタスクはすべて、アカウントアクセスを有効にする前後どちらでも実行できます。
