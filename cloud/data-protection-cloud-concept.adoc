= データ保護とクラウドの概要
:icons: font
:imagesdir: ../media/


[role="lead"]
多くの場合、お客様はクラウドへの移行を開始する際に最初に試すのがデータ保護です。保護は、キーデータの非同期レプリケーションと同様に簡単に行うことも、完全なホットバックアップサイトとして複雑にすることもできます。データ保護は、主に使い慣れた NetApp SnapMirror テクノロジに基づいています。
