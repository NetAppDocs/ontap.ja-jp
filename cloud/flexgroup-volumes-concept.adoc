= FlexGroup ボリューム
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexGroup ボリュームを使用すると、実質的に無制限のサイズの単一ボリュームをアプリケーションに提供できます。Cloud Volumes ONTAP ボリュームは FlexGroup でサポートされているため、 Cloud Volumes ONTAP に FlexGroup ボリュームを導入できます。
