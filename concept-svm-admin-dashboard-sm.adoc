= ONTAP System Manager SVM管理者ダッシュボードについて学ぶ
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.19.1以降、SVM管理者は専用のSystem Managerダッシュボードと[ストレージ] > [ボリューム]ページを使用して、SVMを監視および管理できます。これは、クラスタ管理者が各テナントにSVMダッシュボードを提供できる、マルチテナントONTAP環境で特に役立ちます。

クラスタ管理者は、SVM管理者にログイン認証情報を提供します。SVM管理者は、SVMダッシュボードおよび*ストレージ > ボリューム*ページで詳細情報とパフォーマンスを確認できます。ダッシュボードには、SVMプロトコル、ネットワークインターフェース、詳細情報、およびパフォーマンス指標が表示されます。ボリュームページには、SVMボリュームに関する情報が表示され、クラスタ管理者によって許可されている場合は、SVM管理者が新しいボリュームを追加することもできます。
