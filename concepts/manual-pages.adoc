= ONTAPコマンド リファレンス
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
各主要ONTAPリリースでは、一般的に使用可能な CLI コマンド（ONTAPマニュアル ページまたは man ページ）がlink:https://docs.netapp.com/us-en/ontap-cli/["ONTAP のコマンド リファレンス"^]にバンドルされています。これらのコマンド リファレンスでは、各ONTAPリリースにおける CLI コマンドの使用方法について説明しています。

ONTAPコマンドラインで `man`コマンドを実行すると、マニュアルページも表示されます。link:https://docs.netapp.com/us-en/ontap-cli/man.html["ONTAPコマンド リファレンス"^]の `man`の詳細を確認してください。
