= ONTAPスナップショット リザーブを変更する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
スナップショットがアクティブ ファイル システム用に予約されたスペースを使用しないようにするために、より大きなSnapshot リザーブを設定することを推奨します。スナップショット用のスペースがそれほど必要でなくなった場合は、Snapshot リザーブを減らすことができます。

.手順
. Snapshotリザーブを変更します：
+
`volume modify -vserver _SVM_ -volume _volume_ -percent-snapshot-space _snap_reserve_`

+
 `volume modify`の詳細については、link:https://docs.netapp.com/us-en/ontap-cli/volume-modify.html["ONTAPコマンド リファレンス"^]をご覧ください。

+
次の例では、 `vol1`のスナップショット リザーブを10パーセントに設定します：

+
[listing]
----
cluster1::> volume modify -vserver vs0 -volume vol1 -percent-snapshot-space 10
----
