= ONTAPスナップショットディスク消費量を監視する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
 `df`コマンドを使用して、スナップショットのディスク消費量を監視できます。このコマンドは、アクティブ ファイル システムとスナップショット リザーブの空き容量を表示します。

.手順
. スナップショットのディスク消費量を表示します： `df`
+
次の例は、スナップショットのディスク消費量を示しています：

+
[listing]
----
cluster1::> df
Filesystem          kbytes  used    avail  capacity
/vol/vol0/          3000000 3000000 0       100%
/vol/vol0/.snapshot 1000000 500000  500000   50%
----


この手順で説明されているコマンドの詳細については、link:https://docs.netapp.com/us-en/ontap-cli/["ONTAPコマンド リファレンス"^]を参照してください。
