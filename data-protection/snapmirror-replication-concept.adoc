= SnapMirror ボリュームのレプリケーションについて
:icons: font
:imagesdir: ../media/


[role="lead"]
従来、 ONTAP のレプリケーションテクノロジは、ディザスタリカバリ（ DR ）とデータアーカイブのニーズに対応してきました。ONTAP 9.3 では、これらのテクノロジが融合し、同じデスティネーションボリュームでディザスタリカバリとアーカイブを設定できるようになりました。
