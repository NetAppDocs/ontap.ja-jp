= ローカル階層への容量の追加（アグリゲートの拡張）のワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ローカル階層に容量を追加（アグリゲートを拡張）するには、最初に追加するローカル階層を特定し、必要に応じて新しいストレージの容量を決定し、新しいディスクを設置し、ディスク所有権を割り当て、新しいRAIDグループを作成する必要があります。

容量を追加するには、System ManagerまたはCLIを使用します。
