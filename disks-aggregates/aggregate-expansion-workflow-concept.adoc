= ローカル階層に容量を追加する（アグリゲートの拡張）ワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ローカル階層に容量を追加（アグリゲートを拡張）するには、まずどのローカル階層に追加するかを特定し、必要な新しいストレージ容量を決定し、新しいディスクを取り付けてディスク所有権を割り当て、必要に応じて新しいRAIDグループを作成する必要があります。

容量を追加するには、System ManagerまたはCLIを使用します。
