= ONTAPローカル層に容量を追加するワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ローカル ティアに容量を追加するには、最初に追加するローカル ティアを特定し、必要な新しいストレージの容量を決定し、新しいディスクをインストールし、ディスクの所有権を割り当て、必要に応じて新しいRAIDグループを作成する必要があります。

System Manager または ONTAP CLI のいずれかを使用して容量を追加できます。
