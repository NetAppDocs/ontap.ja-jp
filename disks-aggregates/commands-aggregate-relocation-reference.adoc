= アグリゲートの再配置用のコマンド
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP には、 HA ペアでアグリゲートの所有権を切り替えるための固有のコマンドが用意されています。

|===


| 状況 | 使用するコマンド 


 a| 
アグリゲートの再配置プロセスを開始する
 a| 
`storage aggregate relocation start`



 a| 
アグリゲートの再配置プロセスを監視する
 a| 
`storage aggregate relocation show`

|===
