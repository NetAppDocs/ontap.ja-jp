= アグリゲートに配置するボリュームを決定します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
再配置やオフライン化など、アグリゲートに対する処理を実行する前に、アグリゲートに配置されているボリュームを確認しなければならないことがあります。

.手順
. アグリゲート上のボリュームを表示するには、と入力します
+
volume show -aggregate aggregate-name 」の形式で指定します

+
指定したアグリゲート上にあるすべてのボリュームが表示されます。
