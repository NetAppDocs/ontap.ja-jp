= ONTAPローカル層のメディアコストを設定する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.11.1 以降では、System Manager を使用してローカル階層のメディアコストを設定できます。

.手順
. System Managerで、*Storage > Tiers*をクリックし、目的のローカル階層タイルで*Set Media Cost*をクリックします。
. 比較を有効にするには、*アクティブな階層と非アクティブな階層*を選択します。
. 通貨と金額を入力します。


メディア コストを入力または変更すると、すべてのメディア タイプに変更が適用されます。
