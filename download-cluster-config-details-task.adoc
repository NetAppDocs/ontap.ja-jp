= クラスタ構成の詳細のダウンロード
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.11.1以降では、クラスタ構成の詳細をダウンロードできます。

.手順
. System Manager で、* クラスタ > 概要 * に移動します。
. image:icon-more-kebab-blue-bg.gif["その他のオプションアイコン"]をクリックすると、アクションのリストが表示されます。
. *設定のダウンロード*をクリックします
. HAペアを選択します。
. *ダウンロード*をクリックします。
