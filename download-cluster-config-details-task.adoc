= クラスタ構成の詳細をダウンロードします
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.11.1以降では、クラスタ構成の詳細をダウンロードできます。

.手順
. System Managerで、「* Cluster（クラスタ）」>「Overview（概要）」*に移動します。
. クリックする image:icon-more-kebab-blue-bg.gif["その他のオプションアイコン"] と、アクションのリストが表示されます。
. [Download Configuration*](構成のダウンロード)をクリックします
. HAペアを選択します。
. [ * ダウンロード ] をクリックします。
