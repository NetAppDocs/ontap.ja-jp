= ローカル階層のRAID構成の編集
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.11.1以降では、ローカル階層のRAID構成を変更できます。

.手順
. System Managerで、*Storage > Tiers*をクリックします。
. ローカル ティアの名前の横にあるimage:icon_kabob.gif["メニューオプションアイコン"]をクリックします。
. *RAID 構成の編集*を選択します。
. *RAID構成の編集*ページが表示されます。任意のフィールドを変更できます：
+
--
.. ディスク タイプを選択します。
+
RAIDタイプが混在するローカル階層の場合、このフィールドは表示されません。

.. RAID タイプを選択します。
.. RAIDグループ サイズを選択します。


--



NOTE: RAIDタイプを変更した場合、RAIDグループ サイズは変更できません。また、その逆も同様です。両方のフィールドを変更するには、一方のフィールドを変更して設定を保存してから、もう一方のフィールドでこの編集プロセスを繰り返します。

.関連情報
link:disks-aggregates/aggregate-creation-workflow-concept.html["ローカル階層を追加（作成）する"]を参照してローカル ティアを作成してください。
