= NetApp ElementソフトウェアとONTAPのレプリケーション - 概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapMirrorを使用してElementボリュームのスナップショットをONTAP宛先に複製することで、Elementシステムにおけるビジネス継続性を確保できます。Elementサイトで災害が発生した場合は、ONTAPシステムからクライアントにデータを提供した後、サービスが復旧したらElementシステムを再アクティブ化できます。

ONTAP 9.4以降では、ONTAPノードで作成されたLUNのスナップショットをElementシステムに複製できます。Elementサイトの停止中にLUNを作成したか、LUNを使用してONTAPからElementソフトウェアにデータを移行している可能性があります。

link:https://docs.netapp.com/us-en/element-software/storage/concept_snapmirror_overview.html["NetApp Element ソフトウェアと ONTAP のレプリケーションの設定"^]。
