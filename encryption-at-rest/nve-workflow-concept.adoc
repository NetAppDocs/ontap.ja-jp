= ONTAP NetApp Volume Encryption ワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ボリューム暗号化を有効にする前に、キー管理サービスを設定する必要があります。新しいボリュームと既存のボリュームのいずれでも暗号化を有効にできます。

image:nve-workflow.gif["NetAppボリューム暗号化ワークフロー。ボリュームの暗号化を有効にする手順を示します。"]

link:../encryption-at-rest/install-license-task.html["VEライセンスをインストールする必要があります"]NVEでデータを暗号化する前に、キー管理サービスを設定する必要があります。ライセンスをインストールする前に、link:cluster-version-support-nve-task.html["ONTAPバージョンがNVEをサポートしているかどうかを確認する"]必要があります。
