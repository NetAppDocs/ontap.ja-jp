= ONTAP クラスタに FabricPool ライセンスをインストールする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Cloud Tieringは、2026年4月26日をもって購入およびライセンス更新の提供を終了します。NetApp Cloud Tieringをご利用の場合、NetAppは、NetAppの担当者と協力して既存の階層化ライセンスをONTAP FabricPoolライセンスに移行することをお勧めします。

既存のお客様は、サブスクリプションまたはライセンス契約が満了するまで、NetApp Cloud Tieringを引き続き使用し、サポートを受けることができます。サブスクリプションの有効期限が切れると、NetApp Cloud Tieringの機能やサポートにアクセスできなくなります。

現在 Cloud Tiering ライセンスをご利用の場合は、 Cloud Tiering ライセンスの有効期限が切れるか、最大容量に達するまでは影響を受けません。ライセンスを更新する必要がある場合、またはクラウドへのデータ階層化機能が中断されないようにするために、それよりも前に NetApp にお問い合わせください。

link:https://docs.netapp.com/us-en/data-services-cloud-tiering/task-licensing-cloud-tiering.html["NetApp Cloud Tiering ライセンスの変更の詳細"^]。

System Managerを使用してFabricPoolライセンスをダウンロードしてアクティブ化できます。

FabricPoolライセンスはクラスタ規模のライセンスです。クラスタ内のFabricPoolに関連付けられているオブジェクト ストレージ用に、必要な容量を購入します。クラスタ全体での使用量がこの容量を超えることはできません。ライセンスの使用量の制限を増やす必要がある場合は、営業担当者にお問い合わせください。

FabricPoolライセンスには、無期限のライセンスとタームベース（1年または3年）のライセンスがあります。

既存のクラスター構成に対する初回のFabricPool注文では、10TBの無料容量を含む期間ベースのFabricPoolライセンスを利用できます。永久ライセンスでは、無料容量は提供されません。クラウド階層にNetApp StorageGRIDまたはONTAP S3を使用する場合、ライセンスは必要ありません。Cloud Volumes ONTAPでは、使用しているプロバイダーに関係なく、FabricPoolライセンスは必要ありません。

.開始する前に
NetApp Support Site から NetApp ライセンスファイル（NLF）を取得するには、クラスタ UUID が必要です。

.手順
以下のタスクを実行する場合、System Managerを使用してクラスタにライセンス ファイルをアップロードする必要があります。

. NetApp ライセンス ファイル（NLF）をlink:https://mysupport.netapp.com/site/global/dashboard["NetAppサポート サイト"^]からFabricPoolライセンス用にダウンロードします。
. System Managerで次の操作を実行して、FabricPoolライセンスをクラスタにアップロードします。
+
.. *クラスター > 設定* ペインの *ライセンス* カードで、image:icon_arrow.gif["矢印アイコン"]をクリックします。
.. *ライセンス*ページで、image:icon_add.gif["アイコンを追加"]をクリックします。
.. *[ライセンスの追加]* ダイアログ ボックスで、*[参照]* をクリックしてダウンロードした NLF を選択し、*[追加]* をクリックしてファイルをクラスタにアップロードします。
