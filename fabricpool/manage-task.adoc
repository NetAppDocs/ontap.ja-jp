= Manage FabricPool の概要
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージ階層化のニーズに対応するため、 ONTAP では、ボリューム内のアクセス頻度の低いデータ量の表示、 FabricPool へのボリュームの追加と移動、 FabricPool のスペース使用量の監視、ボリュームの階層化ポリシーや階層化の最小クーリング期間の変更が可能です。
