[role="lead"]
REST API ログには、 System Manager から ONTAP に発行される API 呼び出しが記録されます。このログを使用して、 ONTAP のさまざまな管理タスクを実行するために必要な呼び出しの性質と順序を把握できます。

詳細については、をクリックしてください link:concept_rest_overview.html["こちらをご覧ください"]。
