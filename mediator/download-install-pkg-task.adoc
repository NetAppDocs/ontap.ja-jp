= ONTAPメディエーターのインストールパッケージをダウンロードします
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP Mediator インストール パッケージをダウンロードしてインストールします。

.手順
. ONTAP Mediator ダウンロード ページから ONTAP Mediator インストール パッケージをダウンロードします。
+
https://mysupport.netapp.com/site/products/all/details/ontap-mediator/downloads-tab["ONTAP Mediatorのダウンロードページ"^]

. Mediatorのインストールパッケージを現在の作業ディレクトリに配置してください：
+
[listing]
----
[root@sdot-r730-0003a-d6 ~]# ls ontap-mediator-1.12.0.tgz
----
+
[listing]
----
ontap-mediator-1.12.0.tgz
----
+

NOTE: ONTAP Mediator バージョン 1.4 以前では、インストーラの名前は `ontap-mediator`です。

+
システムがインターネットにアクセスできない場合は、インストーラーが必要なパッケージにアクセスできることを確認してください。

. 必要に応じて、Mediator インストール パッケージをインストール ディレクトリに移動します。
. インストール パッケージを解凍します。
+
`tar xvfz ontap-mediator-1.12.0.tgz`

+
[listing]
----
ontap-mediator-1.12.0/
ontap-mediator-1.12.0/csc-prod-ONTAP-Mediator.pem
ontap-mediator-1.12.0/csc-prod-chain-ONTAP-Mediator.pem
ontap-mediator-1.12.0/tsa-prod-ONTAP-Mediator.pem
ontap-mediator-1.12.0/tsa-prod-chain-ONTAP-Mediator.pem
ontap-mediator-1.12.0/ONTAP-Mediator-production.pub
ontap-mediator-1.12.0/ontap-mediator-1.12.0
ontap-mediator-1.12.0/ontap-mediator-1.12.0.sig.tsr
ontap-mediator-1.12.0/ontap-mediator-1.12.0.tsr
ontap-mediator-1.12.0/ontap-mediator-1.12.0.sig
----
