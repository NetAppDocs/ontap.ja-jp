= ユーザーアカウントのONTAPイベントを管理する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Storage Virtual Machine（SVM）に対してuser-accountイベントが設定されていて、監査が有効になっている場合、監査イベントが生成されます。

イベントID 4720、4722、4724、4725、4726、4738、および 4781 のユーザーアカウントイベントは、ローカル SMB または NFS ユーザーがシステムから作成または削除された場合、ローカルユーザーアカウントが有効化、無効化、または変更された場合、およびローカル SMB ユーザーのパスワードがリセットまたは変更された場合に生成されます。ユーザーアカウントイベントは、 `vserver cifs users-and-groups _<local user>_`および `vserver services name-service _<unix user>_`コマンドを使用してユーザーアカウントが変更された場合に生成されます。

次の例は、ローカル SMB ユーザーが作成されたときに生成された ID 4720 のユーザー アカウント イベントを示しています：

[listing]
----
netapp-clus1::*> vserver cifs users-and-groups local-user create -user-name testuser -is-account-disabled false -vserver vserver_1

次の例は、前の例で作成されたローカルSMBユーザーの名前が変更されたときに生成される、ID 4781のユーザー アカウント イベントを示しています：

----
