= ONTAPセキュリティ トレースの実行方法を学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
セキュリティ トレースを実行するには、セキュリティ トレース フィルターの作成、フィルター条件の検証、フィルター条件に一致する SMB または NFS クライアントでのアクセス要求の生成、および結果の表示が必要です。

セキュリティ フィルターを使用してトレース情報を取得し終わったら、フィルターを変更して再利用したり、不要になった場合は無効にしたりできます。フィルターのトレース結果を表示して分析した後、不要になった場合は削除できます。
