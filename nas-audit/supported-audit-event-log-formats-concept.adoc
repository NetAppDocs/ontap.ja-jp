= サポートされる監査イベントログの形式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
変換された監査イベント・ログでサポートされるファイル形式は 'EVTX および XML 形式です

監査の設定を作成する際には、ファイル形式の種類を指定できます。デフォルトでは、 ONTAP はバイナリログを「 EVTX 」ファイル形式に変換します。
