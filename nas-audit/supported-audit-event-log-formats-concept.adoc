= ONTAP監査イベントログでサポートされている形式について学習します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
変換された監査イベント ログでサポートされているファイル形式は、 `EVTX`および `XML`のファイル形式です。

監査設定を作成する際に、ファイル形式の種類を指定できます。デフォルトでは、ONTAPはバイナリログを `EVTX`ファイル形式に変換します。
