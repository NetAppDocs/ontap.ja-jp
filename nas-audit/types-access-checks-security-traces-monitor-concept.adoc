= ONTAP SVMでセキュリティ トレースが監視するアクセス チェックの種類
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ファイルまたはフォルダへのアクセス チェックは複数の基準に基づいて行われます。セキュリティ トレースは、これらすべての基準に基づいて操作を監視します。

セキュリティ トレースが監視するアクセス チェックの種類は次のとおりです：

* ボリュームとqtreeのセキュリティ形式
* 処理が要求されたファイルやフォルダを含むファイルシステムで有効になっているセキュリティ
* ユーザ マッピング
* 共有レベルの権限
* エクスポートレベルの権限
* ファイルレベルの権限
* Storage-Level Access Guard セキュリティ
