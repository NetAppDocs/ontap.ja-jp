= CNA/UTA2光モジュールをONTAPネットワーク用に変換する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ユニファイド ターゲット アダプタ（CNA / UTA2）用に選択したパーソナリティ モードをサポートするには、そのアダプタで光モジュールを変更する必要があります。

.手順
. カードで現在使用されているSFP+を確認してください。その後、現在のSFP+を、優先パーソナリティ（FCまたはCNA）に適したSFP+に交換してください。
. X1143A-R6アダプタから現在の光モジュールを取り外します。
. 優先して使用するパーソナリティ モード（FCまたはCNA）の光ファイバに適したモジュールを取り付けます。
. 正しい SFP+ がインストールされていることを確認します：
+
[source, cli]
----
network fcp adapter show -instance -node -adapter
----
+
サポートされている SFP+ モジュールおよび Cisco ブランドの銅線（Twinax）ケーブルは、 https://hwu.netapp.com["NetApp Hardware Universe"^]にリストされています。
