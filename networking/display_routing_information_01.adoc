= ONTAPネットワークのルーティング情報を表示する
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
SVMのルートに関する情報を表示できます。

.手順
表示するルーティング情報の種類に応じて、適切なコマンドを入力します。

[cols="40,60"]
|===


| ...に関する情報を表示するには | 入力する内容 


 a| 
SVMごとの静的ルート
 a| 
`network route show`



 a| 
各ルートのLIF（SVMごと）
 a| 
`network route show-lifs`

|===
 `-instance`パラメータを指定することで、各ルートの詳細情報を表示できます。次のコマンドは、cluster-1内のSVM内のスタティック ルートを表示します：

....
network route show
Vserver             Destination     Gateway         Metric
------------------- --------------- --------------- ------
Cluster
                    0.0.0.0/0       10.63.0.1       10
cluster-1
                    0.0.0.0/0       198.51.9.1      10
vs1
                    0.0.0.0/0       192.0.2.1       20
vs3
                    0.0.0.0/0       192.0.2.1       20
....
次のコマンドは、cluster-1のすべてのSVM内の静的ルートと論理インターフェイス（LIF）の関連付けを表示します。

....
network route show-lifs
Vserver: Cluster
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               10.63.0.1               -

Vserver: cluster-1
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               198.51.9.1              cluster_mgmt,
                                                cluster-1_mgmt1,

Vserver: vs1
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               192.0.2.1               data1_1, data1_2

Vserver: vs3
Destination             Gateway                 Logical Interfaces
----------------------  ----------------------  -------------------
0.0.0.0/0               192.0.2.1               data2_1, data2_2
....
 `network route show`および `network route show-lifs`の詳細については、link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=network+route+show["ONTAPコマンド リファレンス"^]をご覧ください。
