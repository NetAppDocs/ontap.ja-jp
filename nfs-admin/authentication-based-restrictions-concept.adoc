= ONTAP NAS SVMの認証ベースの制限について学習します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
認証ベースの制限を使用すると、Storage Virtual Machine（SVM）に接続できるクライアント マシンおよびユーザを指定できます。

ONTAP は、UNIX サーバと Windows サーバの両方からの Kerberos 認証をサポートしています。
