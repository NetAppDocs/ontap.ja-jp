= 認証ベースの制限
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
認証ベースの制限を使用すると、Storage Virtual Machine（SVM）に接続できるクライアントマシンおよびユーザを指定できます。

ONTAP は、 UNIX サーバおよび Windows サーバの両方からの Kerberos 認証をサポートします。
