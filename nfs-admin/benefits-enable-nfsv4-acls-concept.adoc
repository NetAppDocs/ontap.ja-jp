= ONTAP SVM で NFSv4 ACL を有効にするメリットについて学びます
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NFSv4 ACLを有効にすると、多くのメリットがあります。

NFSv4 ACLを有効にする利点は次のとおりです。

* ファイルやディレクトリへのユーザアクセスのより詳細な制御
* NFSセキュリティの強化
* CIFSとの相互運用性の向上
* NFS のユーザあたりの最大グループ数は 16 ではなくなりました
