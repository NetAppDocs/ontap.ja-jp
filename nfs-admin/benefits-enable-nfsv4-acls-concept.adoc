= NFSv4 ACL を有効化する利点
:icons: font
:imagesdir: ../media/


[role="lead"]
NFSv4 ACL を有効化すると多くの利点を得られます。

NFSv4 ACL を有効にする利点は次のとおりです。

* ファイルやディレクトリへのユーザアクセスのより詳細な制御
* NFS セキュリティが向上します
* CIFS との相互運用性の向上
* NFS のユーザあたりの最大グループ数は 16 ではなくなりました
