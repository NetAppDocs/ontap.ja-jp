= ONTAP SVMのNFSv3を有効または無効にする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
 `-v3`オプションを変更することで、NFSv3を有効または無効にできます。これにより、NFSv3プロトコルを使用するクライアントのファイルアクセスが可能になります。デフォルトでは、NFSv3は有効になっています。

.手順
. 次のいずれかを実行します。
+
[cols="20,80"]
|===


| 状況 | コマンドを入力してください... 


 a| 
NFSv3の有効化
 a| 
`vserver nfs modify -vserver vserver_name -v3 enabled`



 a| 
NFSv3を無効にする
 a| 
`vserver nfs modify -vserver vserver_name -v3 disabled`

|===
