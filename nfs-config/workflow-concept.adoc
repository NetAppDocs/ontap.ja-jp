= ONTAP NFSの設定ワークフローについて
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NFSを設定するには、物理ストレージとネットワークの要件を評価して、目的に応じたワークフローを選択します。新規または既存のSVMへのNFSアクセスを設定するか、すでにNFSアクセスの設定が完了している既存のSVMにボリュームまたはqtreeを追加するかによってワークフローが異なります。
