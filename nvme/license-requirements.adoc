= NVMeのライセンス要件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.5以降では、NVMeをサポートするにはライセンスが必要です。NVMeが有効なONTAP 9.4をONTAP 9.5にアップグレードした場合、90日間の猶予期間中にライセンスを取得する必要があります。

ライセンスを有効にするには次のコマンドを使用します。

`system license add -license-code _NVMe_license_key_`
