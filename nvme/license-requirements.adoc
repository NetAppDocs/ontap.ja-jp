= NVMeのライセンス要件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9 .5以降では、NVMeをサポートするにはライセンスが必要です。ONTAP 9でNVMeが有効になっている場合は、ONTAP 9にアップグレードしたあと90日間の猶予期間が与えられます。5.

ライセンスを有効にするには、次のコマンドを使用します。

`system license add -license-code _NVMe_license_key_`
