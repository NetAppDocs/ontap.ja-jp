= ONTAPクラスタとSVMピアリングについて学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ソースクラスタとデスティネーションクラスタ間、およびソースStorage Virtual Machine（SVM）とデスティネーションStorage Virtual Machine（SVM）間にピア関係を作成できます。SnapMirrorを使用してSnapshotをレプリケートする前に、これらのエンティティ間にピア関係を作成する必要があります。

ONTAP 9.3では、クラスタとSVM間にピア関係を設定する方法が簡易化されています。クラスタとSVMのピアリング手順は、ONTAP 9のバージョンごとに異なります。ONTAPのバージョンに応じて、該当する手順に従ってください。

この手順は、System Managerや自動スクリプト ツールではなく、コマンドライン インターフェイス（CLI）を使用して実行します。
