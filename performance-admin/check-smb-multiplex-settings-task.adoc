= ONTAP CIFS/SMBマルチプレックス設定を確認する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
低速なCIFSネットワークが原因でパフォーマンスの問題を引き起こしている場合は、多重化設定を変更して対処することができます。

.手順
. CIFS多重化設定を確認します。
+
`vserver cifs options show -vserver _-vserver_name_ -instance`

. CIFS多重化設定を変更します。
+
`vserver cifs options modify –vserver _-vserver_name_ –max-mpx _integer_`



.例
次の例では、 `SVM1`の最大多重化カウントを255に変更します：

[listing]
----
cluster1::> vserver cifs options modify -vserver SVM1 -max-mpx 255
----
