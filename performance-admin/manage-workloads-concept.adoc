= ワークロードの管理
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP は、ワークロードを管理するための強力なツールセットを提供します。さまざまな方法（コントローラの追加、ボリュームや LUN の移動、フラッシュメディアへのスケールアップなど）でワークロードを他のリソースにシフトできるほか、ストレージサービス品質（ QoS ）を使用して、重要なワークロードのパフォーマンスが競合するワークロードが原因で低下しないようにすることもできます。
