= 日々の監視を実行します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
監視を毎日実行することで、注意が必要なパフォーマンスの問題にすぐに対処することができます。

.手順
. Active IQ Unified Manager UI から * Event Inventory * ページに移動して、現在のイベントと廃止状態のイベントをすべて表示します。
. [*表示*（* View *）]オプションから、を選択します `Active Performance Events` 必要なアクションを決定します。
