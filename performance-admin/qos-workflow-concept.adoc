= ONTAPストレージQoSワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
QoSで管理するワークロードのパフォーマンス要件をすでに把握している場合は、ポリシー グループの作成時にスループット制限を指定できます。把握していない場合は、ワークロードの監視後に制限を指定することもできます。

image:qos-workflow.gif["この図は、次の手順を示しています：1.ポリシーグループに割り当てるストレージオブジェクトを識別します。2.制御および監視するには、スループット制限のあるポリシーグループを作成し、監視のみを行うには、スループット制限のないポリシーグループを作成します。3.ストレージオブジェクトをポリシーグループに割り当てます。4.統計情報を表示してパフォーマンスを監視します。5.必要に応じて、ポリシー設定を調整します。"]
