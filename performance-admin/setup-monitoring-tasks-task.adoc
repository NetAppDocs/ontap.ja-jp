= 基本的なONTAP監視タスクを設定する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
システムを毎日チェックし、週ごとおよび月ごとのパフォーマンス傾向を確立し、潜在的なパフォーマンスの問題に関する通知を受信するためのしきい値を指定することにより、システムのパフォーマンスの問題を監視できます。
