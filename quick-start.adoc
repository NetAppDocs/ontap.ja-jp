= ONTAPのクイック スタート
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
ONTAPを使い始めるには、クラスタをセットアップし、ネットワークを構成し、BMC自動ネットワーク構成を有効にし、ローカル層を作成し、プロトコルを構成し、ストレージをプロビジョニングする必要があります。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"] クラスタをセットアップする
[role="quick-margin-para"]
NetAppでは、System Managerを使用してlink:software_setup/setup-cluster.html["新しいONTAPクラスタをセットアップする"]することを推奨しています。System Managerは、ノード管理IPアドレスの割り当てやクラスタの初期化など、クラスタセットアップのためのシンプルで簡単なワークフローを提供します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["2"]ネットワークを設定する
[role="quick-margin-para"]
link:networking/add_broadcast_domain.html["ブロードキャスト ドメイン"]、link:networking/create_a_subnet.html["サブネット"]、およびオプションでlink:networking/create_ipspaces.html["IPスペース"]を作成して、ネットワークを構成します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["3つ"]BMC自動ネットワーク構成を有効にする
[role="quick-margin-para"]
link:system-admin/enable-sp-bmc-automatic-network-config-task.html["BMC自動ネットワーク構成を有効にする"]各ノードのBMCにIPアドレスを手動で割り当てる必要がなくなります。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["4"] ローカル階層を作成
[role="quick-margin-para"]
link:disks-aggregates/create-aggregates-auto-provision-task.html["ローカル階層を作成する"]ノード内の利用可能なディスクまたはSSDから作成されます。System Managerがハードウェアに基づいて最適な階層構成を自動的に計算します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]プロトコルを設定する
[role="quick-margin-para"]
クラスタ上のlink:software_setup/configure-protocols.html["NAS、SAN、またはNVMeプロトコルを有効にする"]。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["六"]ストレージをプロビジョニングする
[role="quick-margin-para"]
link:concept_nas_provision_overview.html["NAS"]またはlink:san-admin/provision-storage.html["SAN"]構成用のストレージをプロビジョニングします。

.次の手順
* link:insights-system-optimization-task.html["システムの最適化に役立つ分析情報の取得"]。
* link:authentication/index.html["ONTAP管理者認証とロールベースアクセス制御（RBAC）を設定する"]管理者の機能を定義します。
* link:error-messages//configure-ems-notifications-sm-task.html["重要なEvent Management System（EMS）通知を送信するようにONTAPを設定する"]注意が必要なシステムの問題についてすぐに通知されるようにします。
* ONTAPを使用してlink:peering/index.html["災害復旧のためにデータをバックアップする"]する方法とlink:ransomware-solutions/ransomware-overview.html["ランサムウェア攻撃からデータを保護する"]する方法を学習します。
* link:system-admin/autosupport-active-iq-digital-advisor-concept.html["Digital AdvisorとAutoSupportについて学ぶ"]。
* System Managerを使用してlink:task_cp_dashboard_tour.html["クラスタの健全性を監視する"]。
