= ONTAPバージョンに必要な復帰前のタスク
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAPバージョンによっては、リバートプロセスを開始する前に追加の準備タスクを実行する必要がある場合があります。

[cols="2*"]
|===
| ...から戻す場合 | リバートプロセスを開始する前に、次の操作を行ってください... 


| ONTAP 9 のすべてのバージョン  a| 
* link:terminate-smb-sessions.html["継続的に利用できないSMBセッションを終了する"]。
* link:concept_reversion_requirements_for_snapmirror_and_snapvault_relationships.html["SnapMirror関係とSnapVault関係のリバート要件を確認する"]。
* link:task_reverting_systems_with_deduplicated_volumes.html["重複排除ボリュームに十分な空き容量があることを確認する"]。
* link:task_preparing_snapshot_copies_before_reverting.html["スナップショットを準備する"]。
* link:task_setting_autocommit_periods_for_snaplock_volumes_before_reverting.html["SnapLockボリュームの自動コミット期間を時間に設定する"]。
* Metrocluster 構成の場合は、link:task_disable_asuo.html["計画外の自動切り替えを無効にする"]。
* link:arp-warning-clear-revert.html["自律型ランサムウェア対策の異常なアクティビティに関する警告に対処する"]元に戻す前。




| ONTAP 9.18.1  a| 
* ONTAP 9.18.1 アップグレードの一環として ARP の自動有効化が設定されている場合は、link:anti-ransomware-disable-auto-enable.html["無効にする"]必要があります。




| ONTAP 9.17.1  a| 
* SAN に対して ONTAP ARP 機能を有効にしている場合は、link:anti-ransomware-disable-san.html["無効にする"]。




| ONTAP 9.16.1  a| 
* NVMe/TCP 接続に TLS が設定されている場合は、link:task-disable-tls-nvme-host.html["NVMEホスト上のTLS構成を無効にする"]。
* 拡張qtreeパフォーマンス モニタリングが有効になっている場合は、link:disable-extended-qtree-performance-monitoring.html["無効にする"]。
* CORS を使用して ONTAP s3 バケットにアクセスする場合は、link:remove-cors-configuration.html["CORS設定を削除する"]。




| ONTAP 9.14.1 | クライアント接続に対してトランキングを有効にしている場合は、link:remove-nfs-trunking-task.html["NFSv4.1サーバ上のトランキングを無効にする"]。 


| ONTAP 9.12.1  a| 
* NASデータにS3クライアントアクセスを設定している場合、link:remove-nas-bucket-task.html["S3 NAS バケット設定を削除します。"]
* NVMe プロトコルを実行していて、インバンド認証を構成している場合は、link:disable-in-band-authentication.html["インバンド認証を無効にする"]。
* Metrocluster 構成の場合は、link:task-disable-ipsec.html["IPsecを無効にする"]。




| ONTAP 9.11.1 | 自律ランサムウェア保護（ARP）を構成している場合、link:anti-ransomware-license-task.html["ARPライセンスを確認する"]。 


| ONTAP 9.6 | SnapMirror同期関係がある場合は、link:concept_consideration_for_reverting_systems_with_snapmirror_synchronous_relationships.html["リバート用の関係を準備"]。 
|===
