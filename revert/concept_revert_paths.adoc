= サポートされている ONTAP リバートパス
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP ソフトウェアは、現在の ONTAP バージョンより 1 つ前のリリースに直接リバートできます。たとえば、 9.15.1 を実行している場合、 9.13.1 に直接リバートすることはできません。まず 9.14.1 にリバートしてから、 9.14.1 から 9.13.1 への個別のリバートを実行する必要があります。

ONTAP 9.4以前への復元はサポートされていません。サポートされていないONTAPバージョンへの復元は行わないでください。

 `system image show`コマンドを使用して、各ノードで実行されている ONTAP のバージョンを確認できます。

以下のサポートされているリバートパスは、オンプレミス ONTAP リリースのみを対象としています。クラウドでの ONTAP のリバートについては、 https://docs.netapp.com/us-en/cloud-manager-cloud-volumes-ontap/task-updating-ontap-cloud.html#reverting-or-downgrading["Cloud Volumes ONTAPのリバートまたはダウングレード"^]を参照してください。


NOTE: link:https://docs.netapp.com/us-en/ontap-afx/administer/revert-ontap.html["AFXストレージ システム"^] ONTAP ソフトウェアのリバートはサポートされていません。
