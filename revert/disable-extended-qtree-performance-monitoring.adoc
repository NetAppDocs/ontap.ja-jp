= ONTAP 9.16.1から戻す前に、拡張qtreeパフォーマンス監視を無効にします
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.16.1以降では、ONTAP REST APIを使用して、レイテンシメトリックや履歴統計などの拡張qtree監視機能にアクセスできます。拡張qtree監視がいずれかのqtreeで有効になっている場合は、9.16.1からリバートする前に、 `ext_performance_monitoring.enabled`をfalseに設定する必要があります。

link:../volumes/qtrees-partition-your-volumes-concept.html#upgrading-and-reverting["拡張qtreeパフォーマンス監視によるクラスタの復元"]についての詳細をご覧ください。
