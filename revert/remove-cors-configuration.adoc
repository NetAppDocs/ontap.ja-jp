= ONTAP 9.16.1から戻す前にCORS構成を削除します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP S3 バケットにアクセスするために Cross-Origin Resource Sharing（CORS）を使用している場合は、ONTAP 9.16.1 から戻す前にそれを削除する必要があります。

link:../s3-config/cors-integration.html#upgrading-and-reverting["CORS を使用した ONTAP クラスタのリバート"]の詳細をご覧ください。
