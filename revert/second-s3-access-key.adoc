= ONTAP 9.19.1 から元に戻す前に、2 番目の S3 アクセスキーを削除してください
:allow-uri-read: 


[role="lead"]
ONTAP 9.19.1にアップグレードし、いずれかのユーザーに2つ目のS3アクセスキーを追加した場合は、以前のONTAPリリースに戻す前にこれらのキーを削除する必要があります。

.手順
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["S3 アクセスキーを表示"]S3 ユーザーに対して 2 つ目のアクセスキーが追加されているかどうかを確認する。
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["2 つ目のアクセスキーを削除します"]すべてのS3ユーザー向け。
