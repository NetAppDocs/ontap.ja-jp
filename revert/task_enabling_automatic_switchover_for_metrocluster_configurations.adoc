= ONTAP のリバート後に MetroCluster 構成の自動スイッチオーバーを有効にする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP MetroCluster構成をリバートした後、MetroCluster構成が完全に動作可能であることを確認するために、自動計画外スイッチオーバーを有効にする必要があります。

.手順
. 計画外の自動切り替えを有効にする：
+
[source, cli]
----
metrocluster modify -auto-switchover-failure-domain auso-on-cluster-disaster
----
. MetroCluster構成を検証します。
+
[source, cli]
----
metrocluster check run
----
