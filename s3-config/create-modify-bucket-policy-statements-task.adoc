= バケットとオブジェクトストアのサーバポリシーについて
:icons: font
:imagesdir: ../media/


[role="lead"]
S3 リソースへのユーザとグループのアクセスは、バケットとオブジェクトストアのサーバポリシーによって制御されます。ユーザまたはグループの数が少ない場合はバケットレベルでアクセスを制御すれば十分であると考えられますが、ユーザやグループが多数ある場合はオブジェクトストアサーバレベルでアクセスを制御する方が簡単です。
