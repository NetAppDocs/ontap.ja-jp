= ONTAP S3バケットサイズを増減する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
必要に応じて、既存バケットのサイズを増減できます。

.手順
System ManagerまたはONTAP CLIを使用してバケット サイズを管理できます。

[role="tabbed-block"]
====
.System Manager
--
. *Storage > Buckets* を選択し、変更するバケットを見つけます。
. バケット名の横にあるimage:icon_kabob.gif["メニューオプションアイコン"]をクリックし、*編集*を選択します。
. *Edit bucket*ウィンドウで、バケットの容量を変更します。
. *保存*


--
.CLI
--
. バケットの容量を変更します。
+
[source, cli]
----
vserver object-store-server bucket modify -vserver <SVM_name> -bucket <bucket_name> -size {<integer>[KB|MB|GB|TB|PB]}
----
