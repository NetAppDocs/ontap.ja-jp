== ONTAP S3で検証済みの他社製ソリューション

NetAppは、それぞれのパートナーと協力してこれらのソリューションをテストしました。

* Amazon SageMaker
* Apache Hadoop S3Aクライアント
* Apache Kafka
* Commvault（V11）
* コンフルエントカフカ
* NetBackup
* レッドハットキー
* Rubrik
* スノーフレーク
* トリノ
* Veeam（V12）
