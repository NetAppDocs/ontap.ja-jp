= SVMからのNVMeサービスの削除
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
必要に応じて、Storage Virtual Machine（SVM）からNVMeサービスを削除できます。

.手順
. 権限の設定をadvancedに変更します。
+
`set -privilege advanced`

. SVMでNVMeサービスを停止します。
+
`vserver nvme modify -status -admin down`

. NVMeサービスを削除します。
+
`vserver nvme delete`
