= SVM から NVMe サービスを削除します
:icons: font
:imagesdir: ../media/


[role="lead"]
必要に応じて、 Storage Virtual Machine （ SVM ）から NVMe サービスを削除できます。

.手順
. 権限の設定を advanced に変更します。
+
「 advanced 」の権限が必要です

. SVM で NVMe サービスを停止します。
+
「 vserver nvme modify -status-admin down 」を参照してください

. NVMe サービスを削除します。
+
「 vserver nvme delete 」
