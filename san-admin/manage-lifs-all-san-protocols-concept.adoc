= SAN 環境向けの ONTAP LIF について学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP システム内の LUN とネームスペースは、 Storage Virtual Machine （ SVM ）内に格納されます。ネットワークインターフェイス（ LIF ）を使用すると、 SAN クライアントが SVM に接続し、 SVM に含まれる LUN およびネームスペース内のデータにアクセスできます。

* SANをサポートするStorage Virtual Machine（SVM）ごとに管理LIFを1つ設定する必要があります。
* ノードあたり2つのLIFを設定できます。LIFは、iSCSI用のイーサネット ネットワークと分離するために、FCで使用するファブリックごとに1つ必要になります。
* 接続方法としては、直接接続またはイーサネットスイッチの使用がサポートされています。両方の接続タイプに対応するために、LIF を設定する必要があります。
