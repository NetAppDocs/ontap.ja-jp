= FCおよびiSCSI用LUNの管理 - 概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
LUNを作成したら、さまざまな方法で管理できます。例えば、LUNの可用性を制御したり、igroupからLUNのマッピングを解除したり、LUNを削除したり、LUNのサイズを拡張したりできます。
