= FC および iSCSI 用 LUN の管理の概要
:icons: font
:imagesdir: ../media/


[role="lead"]
LUN を作成したら、それらの LUN をさまざまな方法で管理できます。たとえば、 LUN の可用性の制御、 igroup からの LUN のマッピング解除、 LUN の削除、 LUN のサイズの拡張を実行できます。
