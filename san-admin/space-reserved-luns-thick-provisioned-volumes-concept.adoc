:allow-uri-read: 
:icons: font

[role="lead"]
このFlexVol volumeとファイルまたはLUN構成の組み合わせにより、ストレージ効率化テクノロジを使用でき、十分なスペースが事前に割り当てられるため、空きスペースを積極的に監視する必要がありません。
