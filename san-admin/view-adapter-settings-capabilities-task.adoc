= アダプタ設定の確認
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
特定のコマンドを使用して、FC / UTAアダプタに関する情報を表示できます。



== FCターゲット アダプタ

.手順
.  `network fcp adapter show`コマンドを使用してアダプタ情報を表示します： `network fcp adapter show -instance -node node1 -adapter 0a`
+
使用中の各スロットのシステム構成情報とアダプタ情報が出力に表示されます。

+
 `network fcp adapter show`の詳細については、link:https://docs.netapp.com/us-en/ontap-cli/network-fcp-adapter-show.html["ONTAPコマンド リファレンス"^]をご覧ください。





== ユニファイド ターゲット アダプタ（UTA）X1143A-R6

.手順
. ケーブルを接続していない状態でコントローラをブートします。
.  `system hardware unified-connect show`コマンドを実行して、ポート構成とモジュールを確認します。
. ポート情報を確認してから、CNAとポートを設定します。
