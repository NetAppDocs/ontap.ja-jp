= ONTAPによるSANホスト マルチパスのサポート
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAPは、FCホストとiSCSIホストの両方でのマルチパスに非対称論理ユニット アクセス（ALUA）ソフトウェアを使用します。

ONTAP 9.5以降では、非同期名前空間アクセス（ANA）を使用するNVMeホストで、マルチパス高可用性（HA）ペアのフェイルオーバー/ギブバックがサポートされます。ONTAP 9.4では、NVMeはホストからターゲットへのパスを1つしかサポートしないため、アプリケーションホストはHAパートナーへのパスフェイルオーバーを管理する必要があります。

ALUA または ANA をサポートする特定のホスト構成については、ホスト オペレーティング システムの https://mysupport.netapp.com/matrix["NetApp Interoperability Matrix Tool"^]および https://docs.netapp.com/us-en/ontap-sanhost/index.html["ONTAP SAN Host Configuration"]を参照してください。

マルチパスソフトウェア（MPIO（マルチパスI/O）ソフトウェアとも呼ばれます）は、HA構成で推奨されます。マルチパスソフトウェアは、SANホストが複数のパスを介してLUNまたはNVMeネームスペースにアクセスできるようにするために必要です。LUNまたはNVMeネームスペースへのすべてのパスに対して、オペレーティングシステムに単一のディスクとして提示されます。これがないと、オペレーティングシステムは各パスを個別のディスクとして扱い、データ破損につながる可能性があります。

パスが複数あるとみなされるのは、次のいずれかに該当する場合です。

* ホストの1つのイニシエータ ポートをSVMの複数のSAN LIFに接続している場合
* 複数のイニシエータ ポートをSVMの単一のSAN LIFに接続している場合
* 複数のイニシエータ ポートをSVMの複数のSAN LIFに接続している場合


LUNやネームスペースへのアクセスに使用するパスを制限するには、Selective LUN Map、FCスイッチのゾーニング、またはポートセットを使用する必要があります。

== ホストからクラスタ内のノードへの推奨されるパス数

ホストからクラスタ内の各ノードへのパスは8本以下にしてください。また、ホストOSとホストで使用されるマルチパスでサポートできるパスの総数も超えないようにしてください。

クラスタにノードが4つ以上ある場合や、いずれかのノードのSVMで5つ以上のターゲット ポートを使用している場合は、次の方法でノード上のLUNへのアクセスに使用できるパスの数を制限し、推奨される最大数である8個以内になるようにすることができます。

* link:selective-lun-map-concept.html["選択的LUNマップ（SLM）"]
+
SLMは、ホストからLUNへのパスの数を、LUNを所有するノードと、そのノードのHAパートナー上のパスのみに削減します。SLMはデフォルトで有効になっています。クラスタ内のStorage Virtual Machine（SVM）が使用するSLMを介して、各レポートノードに接続するLUNごとに最低2つのパスが必要です。これにより、単一障害点が排除され、システムがコンポーネント障害にも耐えられるようになります。

* link:../san-admin/limit-lun-access-portsets-igroups-concept.html["ポートセット"]
+
ポートセットをSLMと併用することで、特定のターゲットから特定のイニシエータへのアクセスをさらに制限できます。SLMとポートセットを併用する場合、LUNには、そのLUNを所有するノードおよびノードのHAパートナーのポートセットに含まれる一連のLIF経由でアクセス可能になります。

* ホストからのイニシエータグループ（igroup）マッピング
+
igroupを定義してLUNにマッピングし、どのイニシエータがLUNにアクセスできるかを制御できます。

* link:../san-config/fc-fcoe-recommended-zoning-configuration.html["FCスイッチ ゾーニング"]
+
ホストにマルチパス ソリューションがインストールされていない場合、4 台以上のホストが SAN に接続されている場合、またはクラスタ内のノードで SLM が実装されていない場合は、ゾーニング設定を作成する必要があります。
