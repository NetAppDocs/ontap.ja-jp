= LDAP over TLS を設定する
:icons: font
:imagesdir: ../media/


[role="lead"]
LDAP over TLS を設定するには、自己署名ルート CA 証明書のコピーをエクスポートし、エクスポートしたファイルを使用して自己署名ルート CA 証明書をインストールし、 SVM で LDAP over TLS を有効にする必要があります。
