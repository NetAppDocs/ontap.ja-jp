= LDAP over TLSの設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
LDAP over TLSを設定するには、自己署名ルートCA証明書のコピーをエクスポートし、エクスポートしたファイルを使用して自己署名ルートCA証明書をインストールし、SVMでLDAP over TLSを有効にする必要があります。
