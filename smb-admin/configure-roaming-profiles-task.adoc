= Active Directory ユーザーとコンピューター MMC を通じてローミング ONTAP SMB プロファイルを構成する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ユーザがネットワーク上の任意のコンピュータにログオンしたときにそのユーザのプロファイルを自動的に使用可能にするには、[Active Directoryユーザーとコンピュータ]MMCスナップインで移動プロファイルを設定します。Windows Serverで移動プロファイルを設定する場合は、Active Directory管理センターを使用します。

.手順
. Windowsサーバで、[Active Directoryユーザーとコンピュータ]MMC（またはWindows ServerではActive Directory管理センター）を開きます。
. 移動プロファイルを設定するユーザを見つけます。
. ユーザーを右クリックし、*プロパティ* をクリックします。
. *プロファイル* タブで、ユーザーの移動プロファイルを保存する共有へのプロファイル パスを入力し、その後に `%username%`を入力します。
+
たとえば、プロファイル パスは次のようになります： `\\vs1.example.com\profiles\%username%`ユーザーが初めてログインすると、 `%username%`はユーザーの名前に置き換えられます。

+
[NOTE]
====
パス `\\vs1.example.com\profiles\%username%`では `profiles`は、Everyone に対してフル コントロール権限を持つストレージ仮想マシン（SVM）vs1 上の共有の共有名です。

====
. *OK*をクリックします。
