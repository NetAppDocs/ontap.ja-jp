= ONTAP SMBサーバでGPOサポートを有効または無効にする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
CIFSサーバでグループ ポリシー オブジェクト（GPO）のサポートを有効または無効にできます。CIFSサーバでGPOのサポートを有効にすると、グループ ポリシー（CIFSサーバ コンピュータ オブジェクトを含む組織単位に適用されるポリシー）に定義されている該当するGPOがCIFSサーバに適用されます。

[NOTE]
.タスク概要
====
GPOはワークグループ モードのCIFSサーバでは有効にできません。

====
.手順
. 次のいずれかを実行します。
+
|===
| 状況 | コマンドを入力してください... 


 a| 
GPOを有効にする
 a| 
`vserver cifs group-policy modify -vserver _vserver_name_ -status enabled`



 a| 
GPOを無効にする
 a| 
`vserver cifs group-policy modify -vserver _vserver_name_ -status disabled`

|===
. GPO サポートが目的の状態であることを確認します（：） `vserver cifs group-policy show -vserver +vserver_name_`
+
ワークグループ モードの CIFS サーバーのグループ ポリシー ステータスは「`disabled`」と表示されます。



.例
次の例は、Storage Virtual Machine（SVM）vs1でGPOサポートを有効にします。

[listing]
----
cluster1::> vserver cifs group-policy modify -vserver vs1 -status enabled

cluster1::> vserver cifs group-policy show -vserver vs1

            Vserver: vs1
Group Policy Status: enabled
----
