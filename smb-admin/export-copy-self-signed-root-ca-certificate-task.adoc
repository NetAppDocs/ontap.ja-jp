= ONTAP SMB SVMの自己署名ルートCA証明書をエクスポートする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Active Directory通信の保護にLDAP over SSL/TLSを使用するには、まずActive Directory証明書サービスの自己署名ルートCA証明書のコピーを証明書ファイルにエクスポートし、それをASCIIテキスト ファイルに変換する必要があります。ONTAPは、このテキスト ファイルを使用して証明書をStorage Virtual Machine（SVM）にインストールします。

.開始する前に
Active Directory証明書サービスがすでにインストールされ、CIFSサーバが属するドメイン用に設定されている必要があります。Active Directory証明書サービスのインストールと設定の詳細については、Microsoft TechNetライブラリを参照してください。

http://technet.microsoft.com/en-us/library/["Microsoft TechNetライブラリ：technet.microsoft.com/ja-jp/library/"]

.手順
.  `.pem`テキスト形式のドメイン コントローラのルート CA 証明書を取得します。
+
http://technet.microsoft.com/en-us/library/["Microsoft TechNetライブラリ：technet.microsoft.com/ja-jp/library/"]



.終了後の操作
SVMに証明書をインストールします。
