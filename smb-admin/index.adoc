= ONTAP SMBについて学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMBプロトコルでONTAPのファイル アクセス機能を利用できます。CIFSサーバを有効にし、共有を作成し、Microsoftサービスを有効にすることができます。

[NOTE]
====
_SMB_（Server Message Block）は、Common Internet File System（CIFS）プロトコルの最新の方言を指します。ONTAPコマンドライン インターフェイス（CLI）およびOnCommand管理ツールには、引き続き_CIFS_が表示されます。

====
