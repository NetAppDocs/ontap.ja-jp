= ONTAP SMBサーバでDACが無効になっている場合にDAC ACEを含むACLを管理する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ダイナミック アクセス制御ACEが適用されたACLが割り当てられたリソースがある場合にStorage Virtual Machine（SVM）でダイナミック アクセス制御を無効にすると、ダイナミック アクセス制御ACEを削除するまではそのリソースの非ダイナミック アクセス制御ACEを管理できません。

.タスク概要
ダイナミック アクセス制御を無効にした後は、既存のダイナミック アクセス制御ACEを削除するまで、既存の非ダイナミック アクセス制御ACEを削除したり、新しい非ダイナミック アクセス制御ACEを追加したりすることはできません。

これらの手順を実行するには、ACLを管理するために通常使用するツールを使用できます。

.手順
. リソースに適用されているダイナミック アクセス制御ACEを決定します。
. リソースからDynamic Access Control ACEを削除します。
. 必要に応じて、リソースから非ダイナミック アクセス制御 ACE を追加または削除します。
