= NTLM認証
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NTLMクライアント認証は、パスワードをベースとするユーザ固有のシークレットを共有し、チャレンジ-応答プロトコルを使用して行われます。

ユーザーがローカル Windows ユーザー アカウントを使用して SMB 接続を作成すると、CIFS サーバーによって NTLMv2 を使用してローカルで認証が実行されます。
