= NTLM 認証
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NTLM クライアント認証は、パスワードに基づくユーザ固有のシークレットを共有し、チャレンジ - 応答プロトコルを使用して行われます。

ユーザがローカルの Windows ユーザアカウントを使用して SMB 接続を行った場合、認証は、 NTLMv2 を使用する CIFS サーバによってローカルで行われます。
