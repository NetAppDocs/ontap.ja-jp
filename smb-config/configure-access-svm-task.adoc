= SVM への SMB アクセスを設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMB クライアントアクセス用に SVM を設定していない場合は、新しい SVM を作成して設定するか、既存の SVM を設定する必要があります。SMB を設定する場合は、 SVM ルートボリュームへのアクセスを許可し、 SMB サーバを作成し、 LIF を作成し、ホスト名解決を有効にし、ネームサービスを設定し、必要に応じて Kerberos セキュリティの有効化。
