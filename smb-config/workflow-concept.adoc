= SMB の設定ワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMB を設定するには、物理ストレージとネットワークの要件を評価して、目的に応じたワークフローを選択します。新規または既存の SVM への SMB アクセスを設定するか、すでに SMB アクセスの設定が完了している既存の SVM にボリュームまたは qtree を追加するかによってワークフローが異なります。
