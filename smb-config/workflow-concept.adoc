= SMBの設定ワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SMBを設定するには、物理ストレージとネットワークの要件を評価し、目的に応じたワークフローを選択します。新規または既存のSVMへのSMBアクセスを設定するか、すでにSMBアクセスの設定が完了している既存のSVMにボリュームまたはqtreeを追加します。
