= Microsoft Hyper-VおよびSQL Server向けのSMBの設定 - 概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP機能を使用すると、SMBプロトコル経由で2つのMicrosoftアプリケーション（Microsoft Hyper-VおよびMicrosoft SQL Server）に対して無停止運用を有効にできます。

ここで説明する手順は、SMBのノンストップ オペレーションを実装する場合に使用します。想定している状況は次のとおりです。

* SMBプロトコルの基本的なファイル アクセスが設定されている。
* SVMにあるSMB 3.0以降のファイル共有を有効にして次のオブジェクトを格納する。
+
** Hyper-V仮想マシン ファイル
** SQL Serverシステム データベース
