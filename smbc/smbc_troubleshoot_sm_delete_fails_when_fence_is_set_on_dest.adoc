= フェンシングがデスティネーションボリュームに設定されている場合、 SM-BC の SnapMirror 削除処理が失敗します
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


.問題 ：
[role="lead"]
デスティネーションボリュームのいずれかにリダイレクトフェンスが設定されていると、 SnapMirror の削除処理に失敗します。

.解決策
次の操作を実行して、リダイレクションを再試行し、宛先ボリュームからフェンスを削除します。

* SnapMirror が再同期された
* SnapMirror の更新
