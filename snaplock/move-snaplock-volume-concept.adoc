= SnapLock ボリュームを移動
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.8 以降では、 SnapLock ボリュームを同じタイプのデスティネーションアグリゲート（ Enterprise から Enterprise へ、または Compliance to Compliance ）に移動できます。SnapLock を移動するには、 SnapLock セキュリティロールが割り当てられている必要があります。
