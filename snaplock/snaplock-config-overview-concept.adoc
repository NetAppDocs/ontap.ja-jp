= SnapLock を設定します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapLock を使用する前に、SnapLock ライセンスのインストール、コンプライアンスクロックの初期化、SnapLock アグリゲートの作成など、さまざまなタスクを実行してSnapLock を設定する必要があります。
