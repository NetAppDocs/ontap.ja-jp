= ネットワークポートの要件
:allow-uri-read: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
SnapMirror アクティブ同期レプリケーションには、いくつかのネットワークポート要件があります。

例えば、レプリケーションに関与する各ノードは、パートナーノードの状態を監視し、レプリケーション活動を制御する必要があり、データ自体もレプリケーションされなければなりません。さらに、ONTAP Mediator または ONTAP Cloud Mediator は、クォーラムサービスを提供し、フェイルオーバーを支援するために、すべてのノードと通信する必要があります。これらの機能はすべて、エンドポイントとポートの要件が異なります。

ほとんどの場合、SnapMirrorアクティブ同期レプリケーションは、ポートや宛先に基づく制限なしに、内部ネットワーク上で動作します。IPトラフィックがファイアウォール、ルーター、ネットワークトポロジー、またはその他のメカニズムによって制限されている場合、以下の接続要件を検証する必要があります。
