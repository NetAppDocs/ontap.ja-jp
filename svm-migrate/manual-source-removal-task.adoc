= クライアントのカットオーバー後にソース ONTAP SVM を手動で削除する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ソースのクリーンアップを無効にしてSVM移行を実行した場合は、クライアント カットオーバーの完了後にソースSVMを手動で削除できます。

.手順
. ソースのクリーンアップのステータスが準備完了であることを確認します：
+
[source, cli]
----
vserver migrate show
----
. ソースをクリーンアップします。
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
