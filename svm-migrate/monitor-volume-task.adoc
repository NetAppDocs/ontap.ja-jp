= ボリュームの移行を監視
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
を使用してSVMの移行全体を監視することに加えて `vserver migrate show` コマンドを入力すると、SVMに含まれるボリュームの移行ステータスを監視できます。

.手順
. ボリュームの移行ステータスを確認します。
+
`dest_clust> vserver migrate show-volume`
