= ボリュームの移行を監視
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
を使用したSVM移行全体の監視に加えて、を使用したSVM移行全体の監視 `vserver migrate show` コマンドを使用すると、SVMに含まれるボリュームの移行ステータスを監視できます。

.手順
. ボリュームの移行ステータスを確認します。
+
`dest_clust> vserver migrate show-volume`
