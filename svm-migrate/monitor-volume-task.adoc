= ONTAPボリューム移行の監視
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
 `vserver migrate show`コマンドを使用してSVMの移行全体を監視するだけでなく、SVMに含まれるボリュームの移行ステータスを監視することもできます。

.手順
. デスティネーション クラスタで、ボリュームの移行ステータスを確認します：
+
[source, cli]
----
vserver migrate show-volume
----
