= 管理アクティビティの監査設定の管理用コマンド
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
 `security audit`コマンドを使用して、 `audit.log`ファイルに記録される管理アクティビティを管理します。 `cluster log-forwarding`コマンドを使用して、監査ログの転送先を管理します。

|===
| 状況 | 使用するコマンド 


 a| 
デフォルトの設定要求に加えて、ONTAP CLIまたはAPIからの取得要求を監査ログ（ `audit.log`ファイル）に記録することを指定します。
 a| 
`security audit modify`



 a| 
監査ログの設定を表示する
 a| 
`security audit show`



 a| 
クラスタ内の複数のノードからマージされた監査エントリを表示する
 a| 
`security audit log show`



 a| 
監査ログの転送先と、その転送用のセキュリティ対策を指定する
 a| 
`cluster log-forwarding create`



 a| 
監査ログの転送先を変更する
 a| 
`cluster log-forwarding modify`



 a| 
監査ログの転送先を削除する
 a| 
`cluster log-forwarding delete`



 a| 
監査ログに設定されている転送先を表示する
 a| 
`cluster log-forwarding show`

|===
