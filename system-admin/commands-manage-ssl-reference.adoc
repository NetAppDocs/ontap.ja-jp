= SSLの管理用コマンド
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Storage Virtual Machine（SVM）のSSLプロトコルを管理するには、コマンドを使用し `security ssl`ます。

|===
| 状況 | 使用するコマンド 


 a| 
クラスタまたはSVMのSSLを有効にし、デジタル証明書を関連付ける
 a| 
`security ssl modify`



 a| 
クラスタまたはSVMのSSL設定と証明書の名前を表示する
 a| 
`security ssl show`

|===
