= SSL を管理するためのコマンド
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスタ ora Storage Virtual Machine （ SVM ）の SSL プロトコルを管理するには、「 securityssl 」コマンドを使用します。

|===
| 状況 | 使用するコマンド 


 a| 
クラスタまたは SVM の SSL を有効にし、デジタル証明書を関連付けます
 a| 
「セキュリティ SSL の変更」を参照してください



 a| 
クラスタまたは SVM の SSL 設定と証明書の名前を表示する
 a| 
「 securityssl show 」を参照してください

|===
