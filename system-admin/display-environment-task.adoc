= ONTAP環境情報を表示する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
センサーは、システムの環境コンポーネントを監視するのに役立ちます。環境センサーに関して表示できる情報には、種類、名前、状態、値、しきい値の警告などがあります。

.手順
. 環境センサーに関する情報を表示するには、 `system node environment sensors show` コマンドを使用します。
