= ONTAPクラスタ内のノードの属性を表示する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスター内の 1 つ以上のノードの属性（名前、所有者、場所、モデル番号、シリアル番号、ノードの実行時間、ヘルス状態、クラスターへの参加資格など）を表示できます。

.手順
. 指定されたノードまたはクラスター内のすべてのノードの属性を表示するには、 `system node show` コマンドを使用します。


.ノードに関する情報を表示する例
次の例では、node1に関する詳細情報を表示しています。

[listing]
----
cluster1::> system node show -node node1
                          Node: node1
                         Owner: Eng IT
                      Location: Lab 5
                         Model: model_number
                 Serial Number: 12345678
                     Asset Tag: -
                        Uptime: 23 days 04:42
               NVRAM System ID: 118051205
                     System ID: 0118051205
                        Vendor: NetApp
                        Health: true
                   Eligibility: true
       Differentiated Services: false
           All-Flash Optimized: true
            Capacity Optimized: false
                 QLC Optimized: false
    All-Flash Select Optimized: false
 SAS2/SAS3 Mixed Stack Support: none
----
