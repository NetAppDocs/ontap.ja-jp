= ONTAP AutoSupportメッセージの説明を取得する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
受信したAutoSupportメッセージの説明を確認するには、ONTAP Syslog Translatorを使用します。

.手順
. link:https://mysupport.netapp.com/site/bugs-online/syslog-translator["Syslog TranslatorSyslog Translator"^]に移動します。
. **リリース**フィールドに、使用しているONTAPのバージョンを入力します。**検索文字列**フィールドに「callhome」と入力します。*翻訳*を選択します。
. Syslog Translatorに、入力したメッセージの文字列に一致するすべてのイベントがアルファベット順に表示されます。
