= ONTAPにKMIPサーバー用のCA署名付きクライアント証明書をインストールする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
clientおよびserver-caタイプは、証明書がクラスタとKMIPサーバなどの外部キー マネージャとの相互認証に使用されることを指定します。

.タスク概要
KMIPサーバーをクラスターに対してSSLサーバーとして認証するための証明書をインストールします。

.手順
.  `security certificate install`コマンドを `-type server-ca`パラメータとともに使用して、KMIPサーバーの証明書をインストールします。
. プロンプトが表示されたら、証明書を入力し、Enterキーを押します。
+
ONTAP は、将来の参照用に証明書のコピーを保管しておくように通知します。

+
[listing]
----
cluster1::> security certificate install -type server-ca -vserver cluster1

Please enter Certificate: Press <Enter> when done
-----BEGIN CERTIFICATE-----
<certificate_value>
-----END CERTIFICATE-----


You should keep a copy of the CA-signed digital certificate for future reference.

cluster1::>
----
