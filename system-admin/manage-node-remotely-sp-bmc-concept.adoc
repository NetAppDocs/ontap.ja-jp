= SP/BMCを使用したONTAPノードのリモート管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
サービスプロセッサ（SP）またはベースボード管理コントローラ（BMC）と呼ばれるオンボードコントローラを使用して、ノードをリモートで管理できます。このリモート管理コントローラは、現在のすべてのプラットフォームモデルに搭載されています。このコントローラは、ノードの動作状態に関わらず、常に動作し続けます。

プラットフォームSPおよびBMCサポートの詳細については、NetAppサポート サイトのlink:https://mysupport.netapp.com/site/info/sp-bmc["サポートマトリックス"^]を参照してください。
