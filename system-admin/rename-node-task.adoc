= ONTAPノードの名前を変更する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
必要に応じてノードの名前を変更できます。

.手順
. ノードの名前を変更するには、 `system node rename` コマンドを使用します。
+
 `-newname`パラメータはノードの新しい名前を指定します。link:https://docs.netapp.com/us-en/ontap-cli/system-node-rename.html["ONTAPコマンド リファレンス"^]の `system node rename`の詳細をご覧ください。

+
クラスタ内の複数のノードの名前を変更する場合は、各ノードに対して個別にコマンドを実行する必要があります。

+
[NOTE]
====
「`all`」はシステム予約名であるため、ノード名を「`all`」にすることはできません。

====


.ノードの名前を変更する例
次のコマンドは、ノード「`node1`」の名前を「`node1a`」に変更します：

[listing]
----
cluster1::> system node rename -node node1 -newname node1a
----
