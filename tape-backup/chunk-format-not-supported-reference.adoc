= チャンク形式はサポートされていません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Chunk format not supported`

* * 原因 *
+
SMTapeにバックアップイメージが含まれていません。

* * 是正措置 *
+
SMTapeにバックアップイメージが含まれていない場合は、SMTapeバックアップを含むテープを使用して処理を再試行します。
