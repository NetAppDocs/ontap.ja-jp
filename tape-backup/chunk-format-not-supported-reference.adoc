= チャンクフォーマットはサポートされていません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Chunk format not supported`

* * 原因 *
+
SMTape にバックアップイメージが含まれていません。

* * 是正措置 *
+
SMTape にバックアップイメージが含まれていない場合は、 SMTape バックアップを含むテープを使用して処理を再試行します。
