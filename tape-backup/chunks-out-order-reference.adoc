= Chunks out of order
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Chunks out of order`

* *原因*
+
バックアップ テープが正しい順序でリストアされていません。

* *対処方法*
+
リストア処理を再試行し、正しい順序でテープを装填します。
