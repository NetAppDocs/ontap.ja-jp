= Could not obtain vol ref for Volume volume_name
:allow-uri-read: 
:icons: font

* *メッセージ*
+
`Could not obtain vol ref for Volume vol_name`

* *原因*
+
ボリュームが他の処理で使用されている可能性があるため、ボリューム参照を取得できませんでした。

* *対処方法*
+
あとで処理を再試行します。
