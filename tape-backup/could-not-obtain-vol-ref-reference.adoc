= ボリューム volume_name の vol ref を取得できませんでした
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Could not obtain vol ref for Volume vol_name`

* * 原因 *
+
ボリュームが他の処理で使用されている可能性があるため、ボリューム参照を取得できませんでした。

* * 是正措置 *
+
あとで処理を再試行します。
