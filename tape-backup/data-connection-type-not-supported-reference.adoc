= Data connection type ["NDMP4_ADDR_TCP"|"NDMP4_ADDR_TCP_IPv6"] not supported for ["IPv6"|"IPv4"] control connections
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Data connection type ["NDMP4_ADDR_TCP"|"NDMP4_ADDR_TCP_IPv6"] not supported for ["IPv6"|"IPv4"] control connections`

* *原因*
+
ノードを対象としたNDMPモードでは、確立されたNDMPデータ接続のネットワーク アドレス タイプ（IPv4またはIPv6）がNDMP制御接続と同じである必要があります。

* *対処方法*
+
バックアップ アプリケーションのベンダーにお問い合わせください。
