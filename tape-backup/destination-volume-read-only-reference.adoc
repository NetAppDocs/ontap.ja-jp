= Destination volume is read-only
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Destination volume is read-only`

* *原因*
+
リストア先のパスが読み取り専用です。

* *対処方法*
+
データを別の場所にリストアしてください。
