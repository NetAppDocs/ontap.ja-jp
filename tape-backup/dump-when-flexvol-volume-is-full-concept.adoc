= ONTAP FlexVolボリュームがいっぱいになったときのダンプの動作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
増分ダンプ バックアップ処理を実行する前に、FlexVolに十分な空きスペースを確保する必要があります。

操作が失敗した場合は、FlexVol volumeのサイズを増やすか、スナップショットを削除して、FlexVol volumeの空き容量を増やす必要があります。その後、増分バックアップ操作を再度実行してください。
