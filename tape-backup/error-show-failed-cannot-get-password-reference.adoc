= Error:show failed: Cannot get password for user '<username>'
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Error: show failed: Cannot get password for user '<username>'`

* *原因*
+
NDMP用のユーザ アカウントの設定が完了していません。

* *対処方法*
+
ユーザ アカウントにSSHのアクセス方式を関連付け、認証方法としてユーザ パスワードを指定します。
