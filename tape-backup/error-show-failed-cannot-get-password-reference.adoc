= エラー： show failed ： cannot get password for user '<username>'
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Error: show failed: Cannot get password for user '<username>'`

* * 原因 *
+
NDMP のユーザアカウント設定が完了していません

* * 是正措置 *
+
ユーザアカウントが SSH アクセス方法に関連付けられていて、認証方法がユーザパスワードであることを確認します。
