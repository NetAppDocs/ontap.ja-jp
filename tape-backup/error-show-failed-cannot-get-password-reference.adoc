= エラー：show failed：ユーザ「<username>」のパスワードを取得できません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Error: show failed: Cannot get password for user '<username>'`

* * 原因 *
+
NDMPのユーザアカウント設定が不完全

* * 是正措置 *
+
ユーザアカウントがSSHアクセス方式に関連付けられていること、および認証方式がユーザパスワードであることを確認します。
