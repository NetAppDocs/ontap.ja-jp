= Failed to create snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Failed to create snapshot`

* *原因*
+
ボリュームにはすでに最大数の Snapshot が含まれています。

* *対処方法*
+
いくつかのスナップショットを削除してから、バックアップ操作を再試行してください。
