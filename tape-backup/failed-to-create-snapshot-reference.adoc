= Snapshot を作成できませんでした
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
' スナップショットの作成に失敗しました

* * 原因 *
+
ボリュームにはすでに許容最大数の Snapshot コピーが含まれています。

* * 是正措置 *
+
いくつかの Snapshot コピーを削除してから、バックアップ処理を再試行します。
