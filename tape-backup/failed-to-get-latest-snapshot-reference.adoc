= 最新の Snapshot を取得できませんでした
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Failed to get latest snapshot`

* * 原因 *
+
ボリュームが SnapMirror によって初期化されているために、最新の Snapshot コピーが存在しない可能性があります。

* * 是正措置 *
+
初期化が完了してから再試行してください。
