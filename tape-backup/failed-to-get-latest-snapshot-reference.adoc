= Failed to get latest snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Failed to get latest snapshot`

* *原因*
+
ボリュームがSnapMirrorによって初期化されているため、最新のスナップショットが存在しない可能性があります。

* *対処方法*
+
初期化が完了してから再試行します。
