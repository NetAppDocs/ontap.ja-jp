= 新しいテープをロードできませんでした
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Failed to load new tape`

* * 原因 *
+
テープドライブまたはメディアでエラーが発生しました。

* * 是正措置 *
+
テープを交換して処理を再試行してください。
