= Snapshotをロックできませんでした
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Failed to lock snapshot`

* * 原因 *
+
Snapshotは使用中か削除されています。

* * 是正措置 *
+
Snapshotが別の処理で使用されている場合は、その処理が完了するのを待ってからバックアップを再試行します。Snapshotが削除されている場合は、バックアップを実行できません。
