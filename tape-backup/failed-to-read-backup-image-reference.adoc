= Failed to read backup image
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Failed to read backup image`

* *原因*
+
テープが破損しています。

* *対処方法*
+
テープが破損している場合は、リストア処理を実行できません。
