= バックアップイメージを読み取れませんでした
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Failed to read backup image`

* * 原因 *
+
テープが破損しています。

* * 是正措置 *
+
テープが破損している場合は、リストア処理を実行できません。
