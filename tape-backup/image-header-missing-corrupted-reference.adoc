= Image header missing or corrupted
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Image header missing or corrupted`

* *原因*
+
テープに有効なSMTapeバックアップが含まれていません。

* *対処方法*
+
有効なバックアップが含まれているテープを使用して再試行します。
