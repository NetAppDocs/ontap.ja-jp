= バックアップイメージのチェックサムが無効です
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Invalid backup image checksum`

* * 原因 *
+
テープが破損しています。

* * 是正措置 *
+
テープが破損している場合は、リストア処理を実行できません。
