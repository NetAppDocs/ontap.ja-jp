= バックアップイメージのマジック番号が無効です
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Invalid backup image magic number`

* * 原因 *
+
SMTape にバックアップイメージが含まれていません。

* * 是正措置 *
+
SMTape にバックアップイメージが含まれていない場合は、 SMTape バックアップを含むテープを使用して処理を再試行します。
