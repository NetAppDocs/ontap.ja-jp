= 無効なバックアップイメージマジック番号
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Invalid backup image magic number`

* * 原因 *
+
SMTapeにバックアップイメージが含まれていません。

* * 是正措置 *
+
SMTapeにバックアップイメージが含まれていない場合は、SMTapeバックアップを含むテープを使用して処理を再試行します。
