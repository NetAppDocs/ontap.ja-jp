= Invalid backup image magic number
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Invalid backup image magic number`

* *原因*
+
SMTapeにバックアップ イメージが含まれていません。

* *対処方法*
+
SMTapeにバックアップ イメージが含まれていない場合は、SMTapeバックアップを含むテープを使用して処理を再試行します。
