= Invalid input tape
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Invalid input tape`

* *原因*
+
テープ ヘッダー内のバックアップ イメージのシグネチャが有効ではありません。テープ内のデータが破損しているか、テープに有効なバックアップ イメージが含まれていません。

* *対処方法*
+
有効なバックアップ イメージを使用して、リストア ジョブを再試行します。
