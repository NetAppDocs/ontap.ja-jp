= 無効な入力テープです
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Invalid input tape`

* * 原因 *
+
バックアップイメージのシグネチャがテープヘッダーで無効です。テープ内のデータが破損しているか、テープに有効なバックアップイメージが含まれていません。

* * 是正措置 *
+
有効なバックアップイメージを指定して、リストアジョブを再試行します。
