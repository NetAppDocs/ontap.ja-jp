= 無効な入力テープです
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Invalid input tape`

* * 原因 *
+
テープヘッダー内のバックアップイメージのシグネチャが無効です。テープのデータが破損しているか、テープに有効なバックアップイメージが含まれていません。

* * 是正措置 *
+
有効なバックアップイメージを使用してリストアジョブを再試行します。
