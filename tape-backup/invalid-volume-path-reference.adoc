= ボリュームパスが無効です
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Invalid volume path`

* * 原因 *
+
バックアップ処理またはリストア処理に指定されたボリュームが見つかりません。

* * 是正措置 *
+
有効なボリュームパスとボリューム名を指定してジョブを再試行します。
