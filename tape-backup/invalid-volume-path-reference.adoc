= Invalid volume path
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Invalid volume path`

* *原因*
+
バックアップ処理またはリストア処理に指定されたボリュームが見つかりません。

* *対処方法*
+
有効なボリューム パスとボリューム名を指定してジョブを再試行します。
