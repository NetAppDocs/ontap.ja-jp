= Snapshotの自動削除が原因でジョブが中止されました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Job aborted due to snapshot autodelete`

* * 原因 *
+
ボリュームに十分なスペースがないため、Snapshotの自動削除が実行されています。

* * 是正措置 *
+
ボリューム内のスペースを解放してから、ジョブを再試行してください。
