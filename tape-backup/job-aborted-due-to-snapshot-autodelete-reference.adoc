= Snapshot の自動削除によりジョブが中止されました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Job aborted due to snapshot autodelete`

* *原因*
+
ボリュームに十分なスペースがないため、Snapshotの自動削除がトリガーされました。

* *対処方法*
+
ボリューム内の空きスペースを増やしてから、ジョブを再試行します。
