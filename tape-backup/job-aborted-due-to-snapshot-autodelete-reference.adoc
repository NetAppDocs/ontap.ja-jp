= Snapshot の自動削除が原因でジョブが中止されました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
スナップショットの自動削除により 'Job は中止されました

* * 原因 *
+
ボリュームに十分なスペースがないため、 Snapshot コピーの自動削除を実行できません。

* * 是正措置 *
+
ボリューム内のスペースを解放して、ジョブを再試行します。
