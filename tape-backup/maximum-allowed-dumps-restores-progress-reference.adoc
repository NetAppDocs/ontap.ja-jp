= 実行中のダンプまたはリストアの最大許容数（セッション制限の最大数）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Maximum number of allowed dumps or restores _(maximum session limit)_ in progress`

* * 原因 *
+
最大数のバックアップジョブまたはリストアジョブがすでに実行中です。

* * 是正措置 *
+
現在実行中のジョブがいくつか完了してから、処理を再試行します。
