= テープ読み取り時のメディアエラーです
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Media error on tape read`

* * 原因 *
+
データのリストア元のテープが破損しており、テープに完全なバックアップデータが含まれていない可能性があります。

* * 是正措置 *
+
テープに完全なバックアップが含まれていることがわかっている場合は、リストア処理を再試行します。テープに完全なバックアップが含まれていない場合は、リストア処理を実行できません。
