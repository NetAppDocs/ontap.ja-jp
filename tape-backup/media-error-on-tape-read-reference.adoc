= Media error on tape read
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Media error on tape read`

* *原因*
+
データのリストア元のテープが破損しており、テープに完全なバックアップ データが含まれていない可能性があります。

* *対処方法*
+
テープに完全なバックアップが含まれていることがわかっている場合は、リストア処理を再試行します。テープに完全なバックアップが含まれていなければ、リストア処理は実行できません。
