= テープ書き込み時のメディアエラー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Media error on tape write`

* * 原因 *
+
バックアップに使用するテープが破損しています。

* * 是正措置 *
+
テープを交換し、バックアップジョブを再試行します。
