= Media error on tape write
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Media error on tape write`

* *原因*
+
バックアップに使用するテープが破損しています。

* *対処方法*
+
テープを取り替えて、バックアップ ジョブを再試行します。
