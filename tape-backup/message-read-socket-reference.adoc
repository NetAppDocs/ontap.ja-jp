= 読み取りソケットからのメッセージ： ERROR_STRING
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Message from Read Socket: error_string`

* * 原因 *
+
NDMP 3 ウェイ接続でのリモートテープからのリストア通信でエラーが発生しています。

* * 是正措置 *
+
リモートムーバーへのネットワーク接続を確認します。
