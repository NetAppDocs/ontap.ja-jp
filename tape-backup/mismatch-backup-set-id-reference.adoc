= バックアップセット ID が一致しません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Mismatch in backup set ID`

* * 原因 *
+
テープの変更時に装填されたテープが、バックアップセットに含まれるテープではありません。

* * 是正措置 *
+
正しいテープを装填して、ジョブを再試行します。
