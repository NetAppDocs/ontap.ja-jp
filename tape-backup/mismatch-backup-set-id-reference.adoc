= Mismatch in backup set ID
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Mismatch in backup set ID`

* *原因*
+
テープ交換時に装填されたテープが、バックアップ セットに含まれるテープではありません。

* *対処方法*
+
正しいテープを装填して、ジョブを再試行します。
