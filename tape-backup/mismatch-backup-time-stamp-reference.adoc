= Mismatch in backup time stamp
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Mismatch in backup time stamp`

* *原因*
+
テープ交換時に装填されたテープが、バックアップ セットに含まれるテープではありません。

* *対処方法*
+
 `smtape restore -h`コマンドを使用して、テープのヘッダー情報を確認します。
