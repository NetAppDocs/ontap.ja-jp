= バックアップタイムスタンプが一致しません
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
「バックアップのタイムスタンプが一致しません」

* * 原因 *
+
テープの変更時に装填されたテープが、バックアップセットに含まれるテープではありません。

* * 是正措置 *
+
テープのヘッダー情報を確認するには 'smtape restore -h コマンドを使用します
