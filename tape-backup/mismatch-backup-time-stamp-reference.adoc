= バックアップタイムスタンプが一致しません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Mismatch in backup time stamp`

* * 原因 *
+
テープの変更時に装填されたテープが、バックアップセットに含まれるテープではありません。

* * 是正措置 *
+
を使用します `smtape restore -h` コマンドを使用して、テープのヘッダー情報を確認します。
