= ndmpd のバージョン番号が無効です : version_number
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`ndmpd invalid version number: version_number`

* * 原因 *
+
指定した NDMP バージョンがストレージシステムでサポートされていません。

* * 是正措置 *
+
NDMP バージョン 4 を指定します。
