= ndmpdのバージョン番号が無効です: VERSION_NUMBER ``
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`ndmpd invalid version number: version_number`

* * 原因 *
+
指定したNDMPバージョンがストレージシステムでサポートされていません。

* * 是正措置 *
+
NDMPバージョン4を指定します。
