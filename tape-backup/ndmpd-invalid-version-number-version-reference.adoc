= ndmpd invalid version number: version_number ``
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`ndmpd invalid version number: version_number`

* *原因*
+
指定したNDMPバージョンがストレージ システムでサポートされていません。

* *対処方法*
+
NDMPバージョン4を指定します。
