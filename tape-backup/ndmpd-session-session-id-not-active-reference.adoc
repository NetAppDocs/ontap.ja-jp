= ndmpdセッションsession_IDがアクティブでない
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`ndmpd session session_ID not active`

* * 原因 *
+
NDMPセッションが存在しない可能性があります。

* * 是正措置 *
+
コマンドを使用し `ndmpd status`て、アクティブなNDMPセッションを表示します。
