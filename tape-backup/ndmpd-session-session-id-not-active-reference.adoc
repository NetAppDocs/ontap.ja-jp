= ndmpd セッション session_ID がアクティブではありません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`ndmpd session session_ID not active`

* * 原因 *
+
NDMP セッションが存在しない可能性があります。

* * 是正措置 *
+
を使用します `ndmpd status` コマンドを使用して、アクティブなNDMPセッションを表示します。
