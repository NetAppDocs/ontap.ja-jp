= ndmpd session session_ID not active
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`ndmpd session session_ID not active`

* *原因*
+
NDMPセッションが存在していない可能性があります。

* *対処方法*
+
 `ndmpd status`コマンドを使用して、アクティブなNDMPセッションを表示します。
