= ネットワーク通信エラー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
「ネットワーク通信エラー」

* * 原因 *
+
NDMP 3 ウェイ接続でのリモートテープとの通信に失敗しました。

* * 是正措置 *
+
リモートムーバーへのネットワーク接続を確認します。
