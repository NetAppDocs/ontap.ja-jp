= Network communication error
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Network communication error`

* *原因*
+
NDMP 3ウェイ接続でのリモート テープとの通信に失敗しました。

* *対処方法*
+
リモート ムーバーとのネットワーク接続を確認します。
