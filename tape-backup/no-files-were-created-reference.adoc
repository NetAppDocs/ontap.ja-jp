= No files were created
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`No files were created`

* *原因*
+
拡張DAR機能を有効にしないで、ディレクトリDARが試行されました。

* *対処方法*
+
拡張DAR機能を有効にしてから、DARを再試行します。
