= ファイルは作成されませんでした
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`No files were created`

* * 原因 *
+
拡張 DAR 機能を有効にしないで、ディレクトリ DAR が試行されました。

* * 是正措置 *
+
拡張 DAR 機能を有効にしてから、 DAR を再試行します。
