= Tape is currently in use by other operations
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Tape is currently in use by other operations`

* *原因*
+
テープ ドライブが別のジョブで使用中です。

* *対処方法*
+
現在アクティブなジョブが完了してから、バックアップを再試行します。
