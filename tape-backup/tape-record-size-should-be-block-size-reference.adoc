= Tape record size should be block_size1 and not block_size2
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Tape record size should be block_size1 and not block_size2`

* *原因*
+
ローカル リストアに指定されたブロック化因数が正しくありません。

* *対処方法*
+
 `block_size1`をブロック係数として復元ジョブを再試行します。
