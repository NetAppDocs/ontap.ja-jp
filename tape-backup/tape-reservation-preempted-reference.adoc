= テープ予約がプリエンプトされました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Tape reservation preempted`

* * 原因 *
+
テープドライブが別の処理で使用されているか、テープが途中で閉じられました。

* * 是正措置 *
+
テープドライブが別の処理で使用されていないこと、およびDMAアプリケーションがジョブを中止していないことを確認してから再試行してください。
