= Tape reservation preempted
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Tape reservation preempted`

* *原因*
+
テープ ドライブが別の処理で使用されているか、テープがすでに閉じられています。

* *対処方法*
+
テープ ドライブが別の処理で使用されていないこと、およびDMAアプリケーションによってジョブが中断されていないことを確認してから、再試行します。
