= テープ予約が優先されました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Tape reservation preempted`

* * 原因 *
+
テープドライブが別の処理で使用されているか、テープがすでに閉じられています。

* * 是正措置 *
+
テープドライブが別の処理で使用されていないこと、および DMA アプリケーションによってジョブが中断されていないことを確認してから、再試行します。
