= Tape write failed - new tape is broken or write protected
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Tape write failed - new tape is broken or write protected`

* *原因*
+
バックアップに使用するテープが破損しているか、テープに書き込み保護がかけられています。

* *対処方法*
+
テープを取り替えて、バックアップを再試行します。
