= tape write failed -新しいテープが壊れているか、書き込み保護されています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Tape write failed - new tape is broken or write protected`

* * 原因 *
+
バックアップに使用されているテープが破損しているか、書き込み保護されています。

* * 是正措置 *
+
テープを取り替えて、バックアップを再試行します。
