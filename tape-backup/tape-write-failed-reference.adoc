= テープ書き込みに失敗しました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Tape write failed`

* * 原因 *
+
バックアップに使用するテープが破損しています。

* * 是正措置 *
+
テープを交換し、バックアップジョブを再試行します。
