= テープの順序が切れています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Tapes out of order`

* * 原因 *
+
リストア処理に使用する一連のテープの最初のテープに、イメージヘッダーがありません。

* * 是正措置 *
+
イメージヘッダーの付いたテープを装填して、ジョブを再試行します。
