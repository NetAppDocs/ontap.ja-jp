= テープが故障しています
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Tapes out of order`

* * 原因 *
+
リストア処理用のテープシーケンスの最初のテープに、イメージヘッダーがありません。

* * 是正措置 *
+
イメージヘッダーを含むテープをロードし、ジョブを再試行します。
