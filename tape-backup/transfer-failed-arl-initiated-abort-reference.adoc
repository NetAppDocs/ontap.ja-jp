= Transfer failed (ARL initiated abort)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Transfer failed (ARL initiated abort)`

* * 原因 *
+
SMTape処理の実行中にアグリゲートの再配置を開始すると、SMTape処理は中止されます。

* * 是正措置 *
+
アグリゲートの再配置処理が終了したあとにSMTape処理を実行します。
