= Transfer failed (CFO initiated abort)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Transfer failed (CFO initiated abort)`

* *原因*
+
CFOアグリゲートのストレージ フェイルオーバー（テイクオーバーとギブバック）処理が原因で、SMTape処理が中止されます。

* *対処方法*
+
CFOアグリゲートのストレージ フェイルオーバーが終了したあとにSMTape処理を実行します。
