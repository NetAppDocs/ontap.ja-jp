= Transfer failed (CFO initiated abort)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Transfer failed (CFO initiated abort)`

* * 原因 *
+
CFOアグリゲートのストレージフェイルオーバー（テイクオーバーとギブバック）処理が原因で、SMTape処理が中止されます。

* * 是正措置 *
+
CFOアグリゲートのストレージフェイルオーバーが終了したあとにSMTape処理を実行します。
