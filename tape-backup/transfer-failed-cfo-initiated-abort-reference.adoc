= 転送に失敗しました（ CFO が中止を開始）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Transfer failed (CFO initiated abort)`

* * 原因 *
+
CFO アグリゲートのストレージフェイルオーバー（テイクオーバーとギブバック）処理が原因で、 SMTape 処理が中止されます。

* * 是正措置 *
+
CFO アグリゲートのストレージフェイルオーバーが終了したあとに SMTape 処理を実行します。
