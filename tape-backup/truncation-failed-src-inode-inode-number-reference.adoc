= src inode <inode 番号 > …一時的に切り捨てが失敗しました
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
`Truncation failed for src inode <inode number>. Error <error number>. Skipping inode.`

* * 原因 *
+
ファイルのリストア時に、ファイルの inode が削除されます。

* * 是正措置 *
+
ボリューム上のリストア処理が完了するまで待機してから、そのボリュームを使用します。
