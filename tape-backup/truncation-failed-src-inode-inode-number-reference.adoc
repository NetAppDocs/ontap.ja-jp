* * メッセージ *
+
src inode <inode 番号 > の切り捨てに失敗しました。ERROR < エラー番号 > 。inodin.. を省略しています

* * 原因 *
+
ファイルのリストア時に、ファイルの inode が削除されます。

* * 是正措置 *
+
ボリューム上のリストア処理が完了するまで待機してから、そのボリュームを使用します。
