= Unable to lock a snapshot needed by dump
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Unable to lock a snapshot needed by dump`

* *原因*
+
バックアップに指定されたSnapshotは利用できません。

* *対処方法*
+
別のスナップショットを使用してバックアップを再試行します。

+
 `snap list`コマンドを使用して、利用可能なスナップショットのリストを表示します。

+
 `snap list`の詳細については、link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=snap+list["ONTAPコマンド リファレンス"^]を参照してください。
