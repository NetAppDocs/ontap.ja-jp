= Unable to locate bitmap files
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Unable to locate bitmap files`

* *原因*
+
バックアップ処理に必要なビットマップ ファイルが削除されている可能性があります。削除されている場合は、バックアップを再開できません。

* *対処方法*
+
バックアップを再度実行します。
