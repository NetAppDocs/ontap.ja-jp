= ビットマップファイルが見つかりません
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Unable to locate bitmap files`

* * 原因 *
+
バックアップ処理に必要なビットマップファイルが削除されている可能性があります。この場合、バックアップを再開できません。

* * 是正措置 *
+
バックアップを再度実行します。
