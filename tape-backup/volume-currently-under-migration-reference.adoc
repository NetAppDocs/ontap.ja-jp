= Volume is currently under migration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Volume is currently under migration`

* *原因*
+
ボリューム移行とSMTapeバックアップは同時に実行できません。

* *対処方法*
+
ボリューム移行が完了してから、バックアップ ジョブを再試行します。
