= Volume is temporarily in a transitional state
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Volume is temporarily in a transitional state`

* *原因*
+
バックアップ対象のボリュームが一時的にマウント解除された状態になっています。

* *対処方法*
+
少し待ってから、再度バックアップを実行します。
