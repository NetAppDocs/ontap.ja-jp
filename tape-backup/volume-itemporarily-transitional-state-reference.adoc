= ボリュームが一時的な状態にあります
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* * メッセージ *
+
'Volume is temporarily in a transitional state' 」という状態になります

* * 原因 *
+
バックアップ対象のボリュームが一時的にマウント解除された状態になっています。

* * 是正措置 *
+
しばらく待ってから、もう一度バックアップを実行してください。
