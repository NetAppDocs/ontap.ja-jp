= Volume not restricted
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Volume not restricted`

* *原因*
+
データのリストア先ボリュームが制限されていません。

* *対処方法*
+
ボリュームを制限して、リストア処理を再試行します。
