= ボリュームはオフライン
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* * メッセージ *
+
`Volume offline`

* * 原因 *
+
バックアップ対象のボリュームがオフラインです。

* * 是正措置 *
+
ボリュームをオンラインにしてから、バックアップを再試行します。
