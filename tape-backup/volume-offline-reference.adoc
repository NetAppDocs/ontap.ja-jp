= ボリュームはオフライン
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *メッセージ*
+
`Volume offline`

* *原因*
+
バックアップ対象のボリュームがオフラインです。

* *対処方法*
+
ボリュームをオンラインに戻してから、バックアップを再試行します。
