= クラスタダッシュボードにパフォーマンスを表示します

[role="lead"]
ダッシュボードを使用すると、追加または移動するワークロードについて、十分な情報に基づいて意思決定を下すことができます。また、ピーク使用時間を確認して、潜在的な変更を計画することもできます。

パフォーマンスの値は 3 秒ごとに更新され、パフォーマンスグラフは 15 秒ごとに更新されます。

.手順
. [ * ダッシュボード * ] をクリックします。
. [* パフォーマンス *] で、間隔を選択します。
