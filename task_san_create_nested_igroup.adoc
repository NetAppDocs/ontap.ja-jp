= ネストされたigroupの作成
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.9.1以降では、他の既存のigroupで構成されるigroupを作成できます。

. System Managerで、*Host > SAN Initiator Groups*をクリックし、*Add*をクリックします。
. igroup の *Name* と *Description* を入力します。
+
この説明はigroupのエイリアスとなります。

. * Storage VM * と * Host Operating System * を選択します。
+

NOTE: ネストされたigroupのOSタイプは作成後に変更することはできません。

. *イニシエーター グループ メンバー*で*既存のイニシエーター グループ*を選択します。
+
*検索*を使用して、追加するイニシエータ グループを検索して選択できます。
