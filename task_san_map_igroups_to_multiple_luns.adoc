= igroup を複数の LUN にマッピングします

:hardbreaks:

[role="lead"]
ONTAP 9.9..1 以降では、複数の LUN に igroup を同時にマッピングできます。

. System Manager で、 * Storage > LUNs * をクリックします。
. マッピングする LUN を選択します。
. [ * 詳細 * ] をクリックし、 [ * イニシエータ・グループへのマップ * ] をクリックします。



NOTE: 選択した igroup が、選択した LUN に追加されます。既存のマッピングは上書きされません。
