= 複数のLUNへのigroupのマッピング
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP 9.9.1以降では、igroupを複数のLUNに同時にマッピングできます。

. System Managerで、*ストレージ > LUN*をクリックします。
. マッピングするLUNを選択します。
. *[詳細]* をクリックし、*[イニシエーター グループにマップ]* をクリックします。



NOTE: 選択したigroupが、選択したLUNに追加されます。既存のマッピングは上書きされません。
