= 推奨される自動更新の管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
自動更新ログには、推奨される更新のリストとそれぞれの詳細（説明、カテゴリ、スケジュールされたインストール日時、ステータス、エラーなど）が表示されます。このログを確認してから、推奨される更新ごとに実行する操作を判断できます。

.手順
. 推奨される更新のリストを表示します。
+
[cols="2"]
|===
| クラスタ設定から表示 | 更新タブから表示 


 a| 
.. *[クラスタ] > [設定]*をクリックします。
.. ONTAPのバージョンに応じて、以下のいずれかを実行します。
+
*** ONTAP 9.15.1 以前の場合は、*自動更新*セクションでimage:../media/icon_kabob.gif["メニューオプションアイコン"]をクリックし、すべての更新を表示するオプションをクリックします。
*** ONTAP 9.16.1以降の場合、*ソフトウェアアップデート*セクションで、image:icon_arrow.gif["矢印アイコン"]を選択します。*その他のすべてのアップデート*ペインの右隅にある*詳細* image:icon_kabob.gif["メニューオプションアイコン"]をクリックし、すべてのアップデートを表示するオプションをクリックします。



 a| 
.. *[クラスタ] > [概要]*をクリックします。
.. *概要*セクションで*詳細*image:../media/icon_kabob.gif["メニューオプションアイコン"]をクリックし、*ONTAP Update*をクリックします。
.. ONTAPのバージョンに応じて、次の手順を実行します。
+
*** ONTAP 9.15.1 以前の場合は、[*ファームウェア アップデート*] をクリックします。
*** ONTAP 9.16.1 以降の場合は、[*その他のすべての更新*] をクリックします。


.. 更新ページで、*詳細* image:../media/icon_kabob.gif["メニューオプションアイコン"]をクリックし、すべての更新を表示するオプションをクリックします。


|===
. 説明の横にあるimage:../media/icon_kabob.gif["メニューオプションアイコン"]をクリックすると、推奨事項に対して実行できるアクションのリストが表示されます。
+
推奨事項（更新）の状態に応じて、次のいずれかの操作を実行できます。

+
[cols="35,65"]
|===


| アップデートがこの状態の場合： | 次の操作を実行できます。 


 a| 
スケジュールされていません
 a| 
*Update*：更新プロセスを開始します。

*Schedule*：更新プロセスを開始する日付を設定できます。

*Dismiss*：リストから推奨事項を削除します。



 a| 
スケジュール済み
 a| 
*Update*：更新プロセスを開始します。

*スケジュールの編集*：更新プロセスを開始する予定日を変更できます。

*スケジュールのキャンセル*：スケジュールされた日付をキャンセルします。



 a| 
解雇された
 a| 
*閉じない*：推奨事項をリストに戻します。



 a| 
適用中またはダウンロード中
 a| 
*Cancel*：更新をキャンセルします。

|===
