= アップグレード後の特定のONTAP構成を確認する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスタの構成が次のいずれかに該当する場合は、ONTAPソフトウェアのアップグレード後に追加の手順が必要になることがあります。

[cols="2*"]
|===
| 確認する項目 | 答えが*はい*の場合は、次の操作を実行します。 


| ONTAP 9.7以前からONTAP 9.8以降へのアップグレードか | xref:../networking/verify_your_network_configuration.html[ネットワークの設定を確認する] xref:remove-ems-lif-service-task.html[EMS 宛先への到達可能性を提供しないネットワーク サービス ポリシーから EMS LIF サービスを削除します。] 


| クラスタはMetroCluster構成に含まれているか | xref:task_verifying_the_networking_and_storage_status_for_metrocluster_post_upgrade.html[ネットワークとストレージのステータスを確認する] 


| SAN構成があるか | xref:task_verifying_the_san_configuration_after_an_upgrade.html[SAN構成を確認する] 


| ONTAP 9.3以前からのアップグレードで、NetApp Storage Encryptionを使用しているか | xref:task_reconfiguring_kmip_servers_connections_after_upgrading_to_ontap_9_3_or_later.html[KMIPサーバ接続を再設定する] 


| 負荷共有ミラーがあるか | xref:task_relocating_moved_load_sharing_mirror_source_volumes.html[移動された負荷共有ミラーのソース ボリュームを再配置する] 


| ONTAP 9.9.1より前のバージョンで作成されたサービス プロセッサ（SP）アクセス用のユーザ アカウントがあるか | xref:sp-user-accounts-change-concept.html[サービス プロセッサにアクセス可能なアカウントの変更を確認する] 
|===
