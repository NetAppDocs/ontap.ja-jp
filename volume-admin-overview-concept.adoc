[role="lead"]
ONTAP 9.7以降では、System Managerを使用して、FlexVolやLUNなどの論理ストレージ、qtree、ストレージ効率化、クォータなどを管理できます。

クラシックSystem Manager（ONTAP 9.7以前でのみ使用可能）を使用している場合は、 https://docs.netapp.com/us-en/ontap-system-manager-classic/online-help-96-97/concept_managing_logical_storage.html["論理ストレージの管理"^]を参照してください
