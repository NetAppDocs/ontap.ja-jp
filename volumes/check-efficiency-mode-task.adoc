= ボリューム効率化モードを確認します
:icons: font
:imagesdir: ../media/


[role="lead"]
AFF ボリュームに対して「 volume-efficiency show 」コマンドを実行すると、効率化が設定されているかどうかを確認し、現在の効率モードを表示できます。

.ステップ
. ボリュームの効率化モードを確認します。
+
`* volume efficiency show -vserver <vserver name> -volume <volume name> -fields -storage-efficiency -mode *`
