= ボリューム効率化モードを確認します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
を使用できます `volume-efficiency-show` AFF ボリュームに対してコマンドを実行して、効率化が設定されているかどうかを確認し、現在の効率化モードを表示します。

.ステップ
. ボリュームの効率化モードを確認します。
+
`volume efficiency show -vserver <vserver name> -volume <volume name> -fields storage-efficiency-mode`
