= ONTAP で FlexVol ボリュームを管理するためのコマンド
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP CLIには、FlexVolを管理するための固有のコマンドが用意されています。必要な処理に応じて、次のコマンドを使用してFlexVolを管理できます。

[cols="2*"]
|===
| 状況 | 使用するコマンド 


 a| 
ボリュームをオンラインにする
 a| 
`volume online`



 a| 
ボリュームのサイズを変更する
 a| 
`volume size`



 a| 
ボリュームに関連付けられているアグリゲートを確認する
 a| 
`volume show`



 a| 
Storage Virtual Machine（SVM）のすべてのボリュームに関連付けられているアグリゲートを確認する
 a| 
`volume show -vserver -fields aggregate`



 a| 
ボリュームのフォーマットを確認する
 a| 
`volume show -fields block-type`



 a| 
ジャンクションを使用してボリュームを別のボリュームにマウントする
 a| 
`volume mount`



 a| 
ボリュームを制限状態にする
 a| 
`volume restrict`



 a| 
ボリュームの名前を変更する
 a| 
`volume rename`



 a| 
ボリュームをオフラインにする
 a| 
`volume offline`

|===
 `volume`の詳細については、link:https://docs.netapp.com/us-en/ontap-cli/search.html?q=volume["ONTAPコマンド リファレンス"^]を参照してください。
