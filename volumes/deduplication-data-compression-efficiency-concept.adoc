= 重複排除、データ圧縮、データ コンパクション、Storage Efficiency
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexVolでは、重複排除、データ圧縮、データ コンパクションを一緒に、または個別に実行して、最適なスペース削減効果を実現できます。重複排除は、重複したデータ ブロックを削除します。データ圧縮は、データ ブロックを圧縮して必要な物理ストレージ量を削減します。データ コンパクションを実行すると、少ないスペースに多くのデータを格納できるようになり、ストレージ効率が向上します。

[NOTE]
====
インライン重複排除やインライン圧縮などのすべてのインライン ストレージ効率機能は、AFFボリュームではデフォルトで有効になっています。

====
