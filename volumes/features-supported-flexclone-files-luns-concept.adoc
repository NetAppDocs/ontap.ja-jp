= FlexCloneファイルとFlexClone LUNでサポートされる機能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone ファイルと FlexClone LUN は、重複排除、 Snapshot コピー、クォータ、 Volume SnapMirror などのさまざまな ONTAP 機能と相互運用できます。

FlexCloneファイルとFlexClone LUNでサポートされる機能は次のとおりです。

* 重複排除
* Snapshotコピー
* アクセス制御リスト
* クォータ
* FlexCloneボリューム
* NDMP
* ボリュームSnapMirror
* コマンド `volume move`
* スペースリザベーション
* HA構成
