= HA構成とFlexCloneファイルおよびFlexClone LUN
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexCloneファイルおよびFlexClone LUN操作は、HA構成でサポートされます。

HAペアでは、テイクオーバー処理またはギブバック処理が進行している間は、パートナー上にFlexCloneファイルまたはFlexClone LUNを作成できません。パートナー上の保留されたブロック共有処理はすべて、テイクオーバー処理またはギブバック処理が完了したあと再開されます。
