= クォータのUNIX名とWindows名のリンク
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
混在環境では、ユーザはWindowsユーザまたはUNIXユーザとしてログインできます。ユーザのUNIX IDとWindows IDが同じユーザを表すことを認識するようにクォータを設定できます。

次に示す条件の両方が満たされると、Windowsユーザ名のクォータはUNIXユーザ名にマッピングされ、UNIXユーザ名のクォータはWindowsユーザ名にマッピングされます。

*  `user-mapping`パラメータは、ユーザーのクォータ ルールで「on」に設定されています。
* ユーザー名は `vserver name-mapping`コマンドでマッピングされています。


マッピングされたUNIX名とWindows名は同一の個人として扱われ、クォータ使用量の算定に使用されます。
