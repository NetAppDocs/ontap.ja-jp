= ボリュームの論理スペースのレポートと適用 - 概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.4以降では、ボリュームで使用されている論理スペースと残りのストレージ スペースの量をユーザに表示することができます。ONTAP 9.5以降では、ユーザによる論理スペースの使用量を制限できます。

論理スペースのレポートと適用は、デフォルトでは無効になっています。

論理スペースのレポートと適用は、次のボリューム タイプでサポートされます。
