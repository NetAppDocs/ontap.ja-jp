= System ManagerでONTAPボリュームを管理する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
System Managerでボリュームのリストを表示し、さまざまな操作を実行してボリュームを管理できます。

次のボリュームなど、一部のボリューム タイプはSystem Managerでは使用できません：

* Vol0
* DELおよびTMPタイプのボリューム
* FlexGroupコンスティチュエント
* MetroCluster構成内の複製ボリューム


.手順
. System Managerで、*Storage > Volumes*をクリックします。
+
ボリュームのリストが表示されます。

. 次の操作を実行できます。


[cols="25,75"]
|===


| このタスクを実行するには... | 次のアクションを実行してください 


 a| 
ボリュームの追加
 a| 
image:icon_add_blue_bg.gif["アイコンを追加"]をクリックします。link:../task_admin_add_a_volume.html["ボリュームの追加"]を参照してください。



 a| 
複数のボリュームを管理する
 a| 
ボリュームの横にあるボックスにチェックを入れます。

* image:icon_delete_with_can_white_bg.gif["削除アイコン"]をクリックして、選択したボリュームを削除します。
* image:icon_protect.gif["保護アイコン"]をクリックして、選択したボリュームに保護ポリシーを割り当てます。
* image:icon-more-kebab-white-bg.gif["その他のアイコン"]をクリックして、選択したすべてのボリュームに対して実行する以下のアクションのいずれかを選択します：
+
** クォータを有効にする
** Take offline
** Move
** Show Deleted Volumes






 a| 
単一のボリュームを管理する
 a| 
ボリュームの横にあるimage:icon_kabob.gif["メニューオプションアイコン"]をクリックし、実行する次のいずれかのアクションを選択します：

* 編集
* Resize（ONTAP 9.10.1以降、オンライン ボリュームとDP FlexVolのみ）
* 削除
* クローン
* Take Offline / Bring Online
* Enable Quota / Disable Quota
* Edit Export Policy
* Edit Mount Path
* Move
* Edit Cloud Tier Settings
* Protect




 a| 
ボリュームの名前を変更する
 a| 
概要ページからボリュームの名前を変更できます。

ボリューム名の横にあるimage:icon-edit-pencil-blue-outline.png["編集アイコン"]をクリックし、ボリュームの名前を変更します。

|===
