= qtree の変更がクォータの概要に与える影響
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
qtree を削除したり、名前やセキュリティ形式を変更したりすると、現在適用されているクォータに応じて、 ONTAP が適用するクォータが変更される場合があります。
