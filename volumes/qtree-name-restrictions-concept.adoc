= qtree 名の制限事項
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
qtree 名の最大文字数は 64 文字です。また、カンマやスペースなどの特殊文字を qtree 名に使用すると、原因で他の機能に関する問題が発生する可能性があるため、使用しないでください。
