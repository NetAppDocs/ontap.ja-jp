= qtree とクォータ
:icons: font
:imagesdir: ../media/


[role="lead"]
クォータを作成する際に、 qtree をターゲットにすることができます。これらのクォータを、 _tree quotas _ と呼びます。特定の qtree に対して、ユーザクォータやグループクォータを作成することもできます。また、 FlexVol ボリュームのクォータは、そのボリュームに含まれる qtree に継承される場合があります。
