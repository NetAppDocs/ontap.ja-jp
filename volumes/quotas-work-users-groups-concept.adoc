= ユーザおよびグループとクォータの概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クォータのターゲットとしてユーザまたはグループを指定できます。クォータを定義するときは、実装上のいくつかの違いを考慮する必要があります。

注意が必要な相違点には次のようなものがあります。

* ユーザまたはグループ
* UNIXまたはWindows
* 特別なユーザとグループ
* 複数のIDが含まれるか


また、環境に応じてユーザのIDを指定する方法もいくつかあります。
