= ボリュームのスペース削減取り消しに関する詳細情報
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]

スペース削減の取り消しを計画、実装するのに役立つ関連リソースがいくつか用意されています。
