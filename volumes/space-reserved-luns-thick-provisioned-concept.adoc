= シックプロビジョニングされたボリュームを持つスペース予約ファイルまたはLUNの構成設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexVolとファイルまたはLUNの設定に関して、いくつかの組み合わせを使用できます。シックプロビジョニング ボリュームをベースとするこの組み合わせでは、Storage Efficiencyテクノロジを使用できます。事前に十分なスペースが割り当てられるため、空きスペースをアクティブに監視する必要はありません。
