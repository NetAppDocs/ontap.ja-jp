= FlexVolの効率化に関する統計の表示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexVolに対して実行される効率化処理の詳細を表示できます。これは、管理プロセスの有効性を評価するため、またはキャパシティ プランニングの一環として実行できます。

.手順
.  `volume efficiency stat`コマンドを使用して、FlexVolボリューム上の効率化操作の統計を表示します。


.例
次のコマンドを使用すると、ボリューム VolA の効率化操作の統計を表示できます：
`volume efficiency stat -vserver vs1 -volume VolA`

[listing]
----
cluster1::> volume efficiency stat -vserver vs1 -volume VolA

               Vserver Name: vs1
                Volume Name: VolA
                Volume Path: /vol/VolA
Inline Compression Attempts: 0
----
 `volume efficiency stat`の詳細については、link:https://docs.netapp.com/us-en/ontap-cli/volume-efficiency-stat.html["ONTAPコマンド リファレンス"^]を参照してください。
